                    <div class="description">With this ESP-12 breakout board you can connect analog RGB ledstrips, floodlights and digital ledstrips to an esp8266 so you can control your lights with WiFi. <br />Because LEDs are energy friendly, it´s better for the environment and it´s more awesome to! :D</div>

                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p><img src="https://cdn.hackaday.io/images/9112271436469315808.gif"></p><p>features:<br>-It has support for 5 and 12V LED strips.<br>-Support for almost all LED strips currently available. For example, but not limited to: WS2811, WS2812, Neopixel, WS2801 and RGB.<br>-A button to turn the EspLight into Access Point mode, so you can connect your phone or laptop with it and you can set up you local wifi router with a web interface.<br>-A programming mode button.<br>-A reset button.<br> <br>The firmware:<br>Because it´s open source, you can make your own firmware for it.<br>If I finish the Kickstarter (https://www.kickstarter.com/projects/fridgefire/esplight-cheap-open-source-wifi-led-strip-controll/) I will add the following features to make it "noob friendly":<br>-A Windows/Linux program to easy install/upgrade the EspLight.</p><p>-An Android app with at the very least:</p><ul><li>Fade color button for a cool fading effect.</li><li>"Rainbow" mode for digital LED strips.</li><li>Dim option.</li><li>Full on.</li><li>A color "wheel", for this I forked this project: <a href="http://trevorshp.com/creations/android_led.htm" target="_blank">http://trevorshp.com/creations/android_led.htm</a></li></ul>
