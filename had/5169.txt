                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">The idea is simple: pile waste wood in tall, long rows, run soaker hoses/perforated pipe along the top, cover with compost-enriched biochar and soil, then only run water (and the occasional urine to replace the nitrogen the rotting wood locks up as it starts composting) as needed.  For the most part, after the second year, it requires little if any watering *AT*ALL*.  For more info, check out the linked sites.  (I won't be doing any gardening this year, but, considering the theme of this year's HaDP, I wanted to give y'all something to think about.)</div>
                </div>

I have some similar low-tech ideas linked through <a href="https://keithfromcanada.wordpress.com/2013/06/18/to-save-ten-million-lives/" target="_blank">this blog post</a>.  If anyone wants to &apos;steal my thunder&apos;, <strong><em>please go ahead!</em></strong></p>
