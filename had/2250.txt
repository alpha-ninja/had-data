                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description"><p align="justify">I always wanted a model train but seeing the astronomical prices I instantly get rid of that idea, until now. </p><p align="justify">This project's goal is to create a cheap, 3D printed, fully automated model railway system which can be built at home. The model railway is built using the cheapest and fewest parts possible. The brain of the project is a Raspberry Pi, so the entire system can be controlled over the internet from any device with netwok access.</p><p align="justify">It's perfect for home modeling enthusiasts, and for learning and teaching electronics, programming, CAD modeling, 3D printing, etc. The whole project is open-source: the schematics, the 3D models, the source codes are all available on GitHub.</p><p align="justify">This project is as simple as possible: no RF, no GPS, no FPGA, no CAN bus; only basic electronics and AVRs, with a little C programming. </p><p align="justify">So will it change the world? Probably not, but we'll see...</p></div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p align="justify">The goal of the project is to create a cheap, simple, open source, automated, 3D printed, connected, DIY model railway system. Here are the details about why can it be useful and how can it help the world.<br></p><h2>THE PROBLEM</h2><p align="justify">There sure is a problem: young students don&apos;t get enough education in schools about engineering. Sure, they have physics class, and one in a million school has &apos;science fairs&apos;, but that&apos;s all. But did you see any microcontrollers? Did you teacher show you how a transistor works? Did you blink a LED with an arduino or raspi in your programming class? Did they teach you to code anything useful? I just finished high school, and I haven&apos;t seen anything like that (inside school I mean, so I had to learn it all by myself). That&apos;s a serious problem. How can children be interested in engineering if they don&apos;t even see anything interesting? How can young students know how awesome it is to build and program something if they don&apos;t experience it? We need more practical education, and we need it cheap, because schools tend to not have too much money to buy any equipment. We have to make learning engineering fun and interesting instead of just teaching meaningless formulas and code snippets.</p><h2>THE SOLUTION</h2><p align="justify">I only have this one project to offer. It&apos;s not too much, but it&apos;s something. What makes a model railway perfect for being and educational equipment? It&apos;s the complexity of the system. While developing this project, I&apos;ve learned a lot about electronics, coding, 3D printing, and even CAD modeling, and I hope it can also help other students. It is affordable for schools, and can be useful from elementary through high school to technical school. The possibilities are endless.</p><h2>THE STANDARD KIT</h2><p align="justify">Since the project is more like a DIY kit with individual parts rather than one single product, I collected some basic parts into one theoretical product called &quot;standard kit&quot;, to be able to compare the prices, and the components. <strong>The raw material cost of the kit is about 93 USD. </strong>(and almost half of that is the cost of the raspi) If that&apos;s not cheap for a starter model railway kit, I don&apos;t know what is. The components needed for the standard kit are listed below, and you can also view a detailed bill of materials in excel format <a href="https://github.com/lszabi/model_railway/blob/master/model_railway_bom.xls" target="_blank">here</a>.</p><p>Contents of the standard kit:</p><ul><li>1x Master station (the one with the raspi and the power supply)</li><li>1x Normal station</li><li>2x Train</li><li>1x Right switch</li></ul><h2>HOW IT WORKS</h2><p>
