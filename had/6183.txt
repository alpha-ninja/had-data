                    <h5>Description</h5>
                    <div class="description">The ESPLamp is a Web (OpenHAB) enabled RGB-LED Desktop Lamp. It will be powered by at least 3* 24Watts of LED. It consists of a ESP8266 Client with the NodeMCU Firmware and an OpenHAB Server for Management.</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>My aim is to build an RGB Desktop Lamp. It will be powered by an ESP8266 and a openHAB Server. It will also double as an Input Device for the openHab Server, allowing it to change the light directly on the device without the need for an Smartphone / Computer.</p><p>The actual switching will be done by CSD18504KCS. Also I´m working on implementing an Relay for the actual LED Power Supply for conserving power.</p>
                                
                            </p>
                        </div>
