                    <h5>Description</h5>
                    <div class="description">Project aim is to study the positive impact of using automatic light control software in an attempt to improve the sleep quality and measure the results. Research the effects of blue light on sleep quality and create a prototype that can be used to aid sleep. Think of f.lux, but for your room!<br /><br />The project consists of the recent, hackaday proclaimed "solution to anything with WiFi", ESP8266, NodeMCU firmware together with server side software that allows the user to attach new lights and control them. The server is capable of setting timers and delays on messages thus allowing for automatic events to be programmed and executed.<br /><br />The original idea behind the project is to use the sleep tracking software, such as Sleep as Android that was used to measure the sleep quality, by using the recorded data and feeding it back into the server for analysis and decision making in an attempt to improve the next sleep cycle and adapt to users sleeping habits.</div>
                </div>

                        <div class="post-content details-content">
                            <p><p>The idea started from being a frequent user of a screen dimming software on my Linux machine and Android phone. Having read some info about the negative effects of light pollution and how it can hinder sleep quality I decided to build something simple to implement the wireless light bulb that can be used to tackle these issues. With the recent ESP8266 module becoming more popular it was a great pick to drive a wireless LED bulb.</p><p>The project is described in great detail in the final report document along with test results and findings. Please, do check it out!</p><p><a href="https://drive.google.com/open?id=0B_HuE5gtF5VOUmM1TWE4T3d3SlU&amp;authuser=0">View the full report on Google Docs!</a><span></span><br> </p>
