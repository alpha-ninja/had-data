                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">To develop any application we need Output device (Display, Relays), Input device (Keypad/ Sensors), Processing (microcontroller), This Open Development platform FastDuino combines small solutions required for all the big problems on board enabling users to develop their prototype circuits at really fast speed by providing commonly required hardware on board and separate connections for each output and input without increasing board cost.<br /><br />With the commonly available arduino it is observed that most of the project require custom design and it is difficult to have shields for all your need shield utilize IO and make difficult for making connections to other parts.<br /><br />FastDuino is solution combining most required parts on boards, such as LCD, GLCD and Protoboard, with separate connections for each device. It also provides separate connections for SPI, I2C, Serial bus interfaces.<br /><br />FastDuino is software compatible with Arduino UNO</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p><strong>Features:</strong></p><ol><li>Low cost</li><li>Compatible with Arduino UNO.</li><li>On board LCD and GLCD provisions.</li><li>Direct Plugin for bluetooth module.</li><li>Direct Plugin for Relay Board</li><li>Power supply input both AC/DC 9V to 16V</li><li>+3.3V output</li><li>Interrupt switch</li><li>Separate IO connectors for SPI, I2C, Serial</li><li>Uses only 6 IO lines to interface GLCD and LCD.</li><li>Add on adapter for GLCD interface.</li><li>Keypad interface using only one IO line.</li><li>On Board Protoboard</li><li>Fulfills all basic needs of development</li><li>Version 1.0 have FT232 as serial converter</li><li>Updating board with ATmega32 for serial conversion and adding screw terminals on proto board.</li></ol>
                                
                            </p>
                        </div>
