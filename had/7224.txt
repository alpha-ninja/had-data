                    <h5>Description</h5>
                    <div class="description">The problem:<br />Some people think we become unsocial because or phones. This device tries to tackle one of the problems they think causes it. The distraction by our phone because we get a notification about something or because we are texting with someone while in a conversation. The problem with this is that we aren't paying attention to the person we are really talking to.<br /><br />The solution:<br />There would never be a real solution because humans can't multitask. But there could be some devices too reduce the attention it takes so it is easier to switch. That is why my device isn't everyone's solution. Because it uses Morse code it needs to be learned first. And a lot of people don't want or have time to learn it. There is still a group of people that may use it because they would like to use it or already know Morse code.<br /><br />The device:<br />The text to Morse code device or TMC device for short convert text send to it via UDP to Morse code and outputs it with a vibration motor.</div>
