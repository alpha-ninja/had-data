                    <div class="description">MeowCAD is an online free and open source electronic design application tool.  It's focus is on schematic and PCB design for electronic circuits.  MeowCAD is the first completely free and open source, software-as-a-service, electronic design tool and provides a free and open source alternative where there wasn't one before.  Ultimately, the value of an online electronics design community comes from it's members. People are encouraged to fork the project, stand up their own servers or just use MeowCAD directly to create their own electronics projects and share with the community.<br /><br />Check it out at <a href='https://meowcad.com' target="_blank">https://meowcad.com</a><br /><br /><br /><br /></div>

                        <div class="post-content details-content">
                            <p><h1><a href="https://meowcad.com">MeowCAD</a></h1><p><a href="https://meowcad.com">MeowCAD is live, check it out!</a><br></p><p>MeowCAD is an online free and open source electronic design application tool that runs completely in your browser.  It&apos;s focus is on schematic and PCB design for electronic circuits.  MeowCAD is running online now and for people who want to <a href="https://meowcad.com">try it out</a>, they can register for an account or go directly to the <a href="https://meowcad.com/sch">schematic capture</a> and <a href="https://meowcad.com/brd">board capture</a> portions to play around.</p><p>You can see other projects that have <a href="https://meowcad.com/explore">been made public</a> and design your own.  MeowCAD has an easy &apos;snapshot&apos; feature to share quick mockups of schematics and circuit boards.  There&apos;s also library import of KiCAD files to import your own libraries for use in MeowCAD.</p><p>To have your board fabricated, you can export your designs made in MeowCAD to Gerber files (there&apos;s a short <a href="https://meowcad.com/blog/2015-06-16">blog post about it</a>).  Export also has experimental features to create KiCAD files and GCode files for isolation routing (all source for <a href="https://github.com/abetusk/pykicad">conversion</a> <a href="https://github.com/abetusk/weakpwh">is</a> <a href="https://github.com/abetusk/gbl2ngc">also</a> free and open source).<br></p><p>MeowCAD is the first completely free and open source, software-as-a-service, electronic design tool and provides a free and open source alternative where there wasn&apos;t one before.  Source is available on GitHub, both the <a href="https://github.com/abetusk/www.meowcad.com">website</a> and the <a href="https://github.com/abetusk/bleepsix">JavaScript web application</a> powering the schematic and board portion.</p><h2>Links</h2><p><a href="https://github.com/abetusk/www.meowcad.com/blob/release/notes/DesignDocument.md">Design Document for MeowCAD</a></p><p><a href="https://github.com/abetusk/www.meowcad.com">MeowCAD website sourcode (on GitHub)</a></p><p><a href="https://github.com/abetusk/bleepsix">Web application for schematic and board capture (bleepsix)</a></p><p><a href="https://github.com/abetusk/bleepsix/wiki/0_Home">Web application (bleepsix) Wiki</a><br></p><h2>License</h2><p>Most of the main source code is under AGPLv3.  Where possible,  3rd party libraries licenses have not been altered.  All 3rd party software has a free license of one sort or another.  Please refer to the above source links for licenses for the respective portions.</p>

                            <h2><a href="/project/7229/log/23008-board-fabrication">Board Fabrication</a></h2>
                            <small class="authorship">2 days ago  &bull;
                            <a class="gray-link" href="/project/7229/log/23008#discussion-list">
                                0 comments
                            </a>
                            </small>
                            <p><h1>PCB Fabrication</h1><p>Ultimately <a href="https://meowcad.com/">MeowCAD</a> is meant to create boards.  The real test is whether you can go from idea to full board.</p><p>I&apos;ve created two functioning boards with MeowCAD.  Both have been designed completely in MeowCAD, schematics and PCBs, sent to OSHPark for creation and reflowed at home with my own reflow oven (a converted convection toaster oven).  You can read a bit about it in a <a href="https://meowcad.com/blog/2015-06-16">blog post I did</a>.</p><p>Both circuits are very simple and are more of a &apos;proof of concept&apos; than anything else.  Remember, I&apos;m still learning electronics.  I built MeowCAD to help me with my electronics education.  Both circuits were my first  time designing and reflowing surface mount components (using 1208 and 0603 parts).</p><h2><a href="https://meowcad.com/project?projectId=1bd24ce3-91d9-4846-b30d-1a8cfacd72d0">8-Bit Heart</a></h2><p>The first board I successfully created through MeowCAD was the &apos;<a href="https://meowcad.com/project?projectId=1bd24ce3-91d9-4846-b30d-1a8cfacd72d0">8-Bit Heart</a>&apos;.  Below is the board in action.  The board itself is about 2 inches square with an ATMega328 directly driving a 4x4 1208 (red) LED matrix.  The battery is a CR2032 pill battery (the kind you find in watches sometimes).  One of the reasons I wanted AdaFruit and Dangerous Prototypes library import was because they both had a nice CR2032 surface mount holder that I wanted to use that wasn&apos;t available through the standard KiCAD libraries.</p><p>The board scrolls a message across the LED display in &apos;ticker tape&apos; fashion.  Below is a video of it in action:</p><iframe style="width: 500px; height: 281px;" width="500" height="281" src="//www.youtube.com/embed/0BI4rqI0vt0" frameborder="0" allowfullscreen=""></iframe><p>Source is available on my <a href="https://github.com/abetusk/8bitheart">GitHub page</a>.</p><h2><a href="https://meowcad.com/project?projectId=12b1cb79-9d14-4319-a8cc-878b5ee93019">h8ARt</a></h2><p>The second board was a smaller one.  I originally wanted to create an earring but I think the board, while small, is still too large.</p><p>The &apos;<a href="https://meowcad.com/project?projectId=12b1cb79-9d14-4319-a8cc-878b5ee93019">h8ARt</a>&apos; board uses an attiny13 driving a 3x3 0603 (red) LED matrix via <a href="https://en.wikipedia.org/wiki/Charlieplexing">Charlieplexing</a>.  The battery is mounted on the back and is a CR1220.  It also displays a message in &apos;ticker tape&apos; style though the font is extremely hard to read as 3x3 is not that legible.  I originally had the source in C but quickly ran into the 1k memory limit of the attiny13 and had to go to assembly to fit all the functionality I wanted in it.  Below is a video of it in action.</p><iframe style="width: 500px; height: 281px;" width="500" height="281" src="//www.youtube.com/embed/Gg_DkGkVkSI" frameborder="0" allowfullscreen=""></iframe><p>Source is also available on my <a href="https://github.com/abetusk/h8ARt">GitHub page</a>.</p><h2>Success!</h2><p>The circuits are simple but I&apos;m very happy with how they turned out.  MeowCAD can be used to actually fabricate boards!</p>
                                
                            </p>
                        </li>

                                0 comments
                            </a>
                            </small>
                            <p><h1>GCode Export</h1><p>MeowCAD has the ability to export to GCode as well.  The feature is still experimental and the GCode itself might be a little &apos;quirky&apos; but the basics are there.</p><p>The main program that does the conversion is the <a href="https://github.com/abetusk/gbl2ngc">gbl2ngc</a> which takes in a Gerber file and produces a GCode file.  The heavy lifting is done by the excellent <a href="http://www.angusj.com/delphi/clipper.php">clipperlib</a> library (by Angus Johnson).  MeowCAD uses this during it&apos;s export process via the <a href="https://github.com/abetusk/www.meowcad.com/blob/release/cgi/downloadProject.py">downloadProject.py</a> file.  downloadProject.py calles gbl2ngc externally and collects the result into a final ZIP file to give back to the user.<br></p><p>I have a <a href="http://mechaelephant.com/ngc_view?id=5491563e-5925-4f72-8209-855b578acd8e">GCode viewer</a> running on one of my other servers.  Below is a screenshot of the front copper layer (there are others):</p><p><img src="https://cdn.hackaday.io/images/7185971439710758517.png"></p>

                            <p><h1>Eagle to KiCAD Library Conversion</h1><p>MeowCAD uses an older version of KiCAD&apos;s board format and it&apos;s useful for MeowCAD to have a conversion from Eagle the older KiCAD version.  In addition, it&apos;s useful to add some &apos;default&apos; components and footprints from some hobby friendly (and open source hardware friendly) companies like Adafruit, SparkFun and Dangerous Prototypes.</p><p>The main repo that was used was a fork of <a href="https://github.com/DanChianucci/Eagle2Kicad">DanChianucci&apos;s</a> <a href="https://github.com/abetusk/Eagle2Kicad">Eagle2Kicad</a> tool.  I modified this slightly to be more command line friendly.  The source is available at my <a href="https://github.com/abetusk/Eagle2Kicad">Eagle2Kicad</a> GtiHub.</p><p>Converted are available in the <a href="https://github.com/abetusk/www.meowcad.com">MeowCAD</a> website source but can also be found in the <a href="https://github.com/abetusk/Open-Hardware-Libraries">Open-Hardware-Libraries</a> source.</p><p><img src="https://cdn.hackaday.io/images/4751531439710241345.png"></p>
