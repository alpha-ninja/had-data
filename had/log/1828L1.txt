                            <small class="authorship">5 months ago  &bull;
                                <a class="gray-link" href="/project/1828/log/15923#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-15923"><p>Flyver SDK allows you to serve a webpage directly from your smartphone.</p><p>To do so, you must put all your webpage files(.html, .css, .js, images) in the FlyverCore/src/main/assets/webpage/ folder(only plain files, folder structure is not supported for now) and the page will be served on the device's IP address. The files in this folder are deployed automatically with the .apk of the application</p><p>The HTTP server used for this is <a href="https://github.com/NanoHttpd/nanohttpd">NanoHTTPD</a>, which is subclassed in utils/java/co.flyver.utils/NanoHTTPDServer, which implements its serve method</p><p>Any questions? Ask in the comments below.</p><p>This article was originally posted on the <a href="https://github.com/flyver/Flyver-SDK/wiki/-10--Webpage-Server">Flyver Github</a></p><img src="http://flyver.co/wp-content/uploads/bfi_thumb/flyver_04-m3gpueqrum9bsfc44mk2i8tra4wziqqzryxuyp3cmk.jpg"></div>
