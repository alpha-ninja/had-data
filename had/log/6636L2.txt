                            <h2><a href="/project/6636/log/23707-iced-testing">ICEd testing</a></h2>
                            <small class="authorship">12 hours ago  &bull;
                                <a class="gray-link" href="/project/6636/log/23707#discussion-list">

                            <div style="position:relative">
                                <div id="post-body-23707"><p>ICEd is mostly compatible to Lattice ICEstick, same FPGA many I/Os are at same pins, so a lot of code that works on ICEstick would also work on ICEd without changes.</p><p>Examples:<br></p><ol><li>yosys example &quot;rot&quot; compiled with ICEstorm (rot.bin) would blink the RGB LED on ICEd</li><li>j1a.bin from swapforth would work also</li></ol><p>Note when trying j1a.bin then need check the terminal setting because of DTR use for reset. With putty there would be possible no output, using parallax terminal I got response (UART setting 115200)</p><p>Step 1:</p><p>Connect micro USB Cable to ICEd</p><p>Step 2:</p><p>Click detect, make sure FTUSB-0 is selected as Port</p><p><img src="https://cdn.hackaday.io/images/5406101439925799711.png"></p><p>Step 3: Click scan</p><p><img src="https://cdn.hackaday.io/images/3917161439925875160.png"></p><p>Step 4:</p><p>Click on the Yellow cell and select HX1K from drop down</p><p><img src="https://cdn.hackaday.io/images/8885841439925957254.png"></p><p>Step 5: Select say rot.bin from IceStorm project</p><p>can also grab here</p><p><a href="https://github.com/AnttiLukats/ICEd/tree/master/THP2015/IceStorm">https://github.com/AnttiLukats/ICEd/tree/master/THP2015/IceStorm</a><br></p><p><img src="https://cdn.hackaday.io/images/2126381439926046183.png"></p><p>Step 6:</p><p>Click on green arrow, this will load the code into the HX1K configuration RAM</p><p><img src="https://cdn.hackaday.io/images/4013881439926185095.png"></p><p>FPGA on ICEd is configured OK</p><p><img src="https://cdn.hackaday.io/images/3067981439926348134.jpg"></p><p>Both the RGB LED and the small green SMD LED should blink: rot.v does toggle 5 LEDs (on ICEstick) ICEd has those 5 LEDS mapped to RGB LED and that small green one.</p></div>
