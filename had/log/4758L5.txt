                            <small class="authorship">3 days ago  &bull;

                                    0 comments
                                </a>

                            <div style="position:relative">
                                <div id="post-body-23004"><p>To make radio distance any significant distance requires height, because of such things as the <a href="https://en.wikipedia.org/wiki/Fresnel_zone">Fresnel Zone</a>.</p><p>Luckily, in Australia it is a rare working farm that doesn&apos;t have a windmill (a particularly common sight in areas not served by mains water) and/or a dedicated radio tower. This is good, it will let us design a system that can have a high central node and reduce the amount of elevation required for antennas in the paddocks.</p><p>The photo shows the tower that someone will need to climb when we are ready to start extending the range of SentriFarm :-)</p><p><img src="https://cdn.hackaday.io/images/7734501439711573900.jpg"></p></div>
                            </div>
                        </li>
                    
                        <li>
                            <h2><a href="/project/4758/log/23302-state-of-play-august-17">State of Play August 17</a></h2>
                            <small class="authorship">2 days ago  &bull;
                                <a class="gray-link" href="/project/4758/log/23302#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-23302"><p>We will be uploading our video Real Soon Now.<br></p><p>To date, we have assembled circuits to build four configuration combinations of sensor module, although until further components arrive we can presently only have one pair of any configuration combination operational at a time.</p><p><br>We have working minimum operational software for Linux and for the Arduino framework, and have run a basic telemetry feed overnight between buildings on my properly. </p><p>The main bulk of work (apart from building additional modules so we can have a pair deployed with the solar power to a paddock at the same time as a pair on the workbench for development) remaining is software and antennas : (a) coding in the additional sensors, tuning the protocols, and integration with OpenHAB, and (b) extending the range out to the 10+km the inAir9 radios are advertised as capable of.  The other major task remaining is concluding development of the enhanced rain gauge.<br></p><p>For a full summary of outstanding implementation refer to the <a href="https://github.com/pastcompute/sentrifarm/blob/master/TODO.md">TODO</a> document in github</p></div>
