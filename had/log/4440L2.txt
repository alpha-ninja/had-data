                            <div style="position:relative">
                                <div id="post-body-21160"><p>To legally operate the oGPR in the United States (for non-commercial use), you will need a Technician Class (or better) amateur radio license. The 35 question exam is not difficult if you study. All 426 possible questions are <a href="http://ncvec.org/page.php?id=362">published</a>. Ron Hashiro has a good <a href="http://www.qsl.net/ah6rh/am-radio/license-tech/">study guide</a>. The <a href="http://www.arrl.org/">ARRL</a> has <a href="http://www.arrl.org/getting-your-technician-license">more information</a>. </p></div>

                        <li>
                            <h2><a href="/project/4440/log/21279-rover-test-results-upgrades-planned">Rover Test Results; Upgrades Planned</a></h2>
                            <small class="authorship">a month ago  &bull;
                                <a class="gray-link" href="/project/4440/log/21279#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-21279"><p>The current design performs acceptably on pavement. It moves at about walking speed, which is the targeted 1 m/s.</p><p>It works in fields, but I find the performance to be qualitatively unacceptable.</p><strong></strong><p><strong><em>I am trying to build the least expensive system that will work.</em></strong></p><p>I&apos;m currently using the &quot;<a href="http://www.amazon.com/Wondermotor-Reversible-Electric-Gear-Motor/dp/B00LTM3AT6">Wondermotor 12V DC Reversible Electric Gear Motor 50 RPM</a>&quot; which performs just below that line.</p><p>I am planning on upgrading to <a href="http://www.ampflow.com/wheelmotors.htm">AmpFlow 12.5-inch WheelMotors with E30-150 Motors</a><span class="redactor-invisible-space"> for $298 each.</span></p><p>This is about <em>three times</em> the cost of the current motor/wheel setup which includes: </p><p>$69 motor<br>$9 wheel<br>$8 pillow block bearing<br>$6 axle<br>$14 Lovejoy couplings<br></p><p>But, the AmpFlow WheelMotor negates the need for a custom manufactured mounting assembly, which was fun to make. But, access to CNC mills and plasma cutters is limited and it was a time consuming process.</p><p><em>&quot;[<a href="http://www.ampflow.com/">AmpFlow</a>] motors and controllers are used in diverse applications including robots, 
industrial motion control, factory automation, semiconductor manufacturing, 
	educational projects, and motorized
    scooters, go karts,
    and electric bicycles.</em><em>&quot;</em><span class="redactor-invisible-space"></span></p><p>So, they would appear to be exactly what I need. But, <a href="http://www.ampflow.com/motor_controller.htm">AmpFlow Electronic Motor Controllers</a> are $500!</p><p>After a bit of consideration, I realized that I could use <a href="http://www.aliexpress.com/item/diameter-20mm-SCK1-200A-dc-current-sensor-hall-effect/32254516930.html">Hall Effect DC Current Sensors</a><span class="redactor-invisible-space"> (~$8x2), <a href="http://www.aliexpress.com/store/product/Double-BTS7960-43A-H-bridge-High-power-Motor-Driver-module-smart-car/325585_531569334.html">IBT-3 motor drivers</a> (~$10x2), thermocouples (~$1x2), (plus shipping,) and modified<a href="http://rover.ardupilot.com/"> APM:Rover</a> firmware to achieve my performance targets f<em></em>or about<em> 1/10th the cost</em> of the AmpFlows EMC! APM:Rover firmware could also be modified to allow a &quot;<a href="https://en.wikipedia.org/wiki/Battleshort">Battleshort</a>.&quot; Because sometimes you just <a href="https://www.youtube.com/watch?v=qs-ANd7ZncQ">need more power</a>!<br></span></p></div>
                            </div>
