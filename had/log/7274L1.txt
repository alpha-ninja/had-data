                            <div style="position:relative">
                                <div id="post-body-23313"><p>A 1st prototype has been built. It features a graphic display to interact with, a Bluetooth module, a Li-ion battery and two push buttons to navigate through the menu.<br></p><p><img src="https://cdn.hackaday.io/images/4156761439818068417.jpg" style="width: 622px; height: 471px;" width="622" height="471"></p><p>An Android application has been developed to demonstrate the smartwatch - smartphone communication side. The smartwatch will look for its geographical location to retrieve weather information &amp; will also check if any new messages arrived or if anyone is calling.</p><p>The application has been developed in Java using the Eclipse IDE and the Android SDK tools. The weather information is retrieved by parsing JSON data from openweathermap.org. The website provides useful information related to weather, temperature, humidity, wind speed and so on. </p><p>The smartwatch got it&apos;s first firmware. It was written in Arduino because it&apos;s fast and minimizes development life cycles. The firmware talks with the Android application through the Bluetooth interface. It parses information like weather, messages and caller id.</p></div>

                            <h2><a href="/project/7274/log/23332-braille-display">Braille display</a></h2>
                            <small class="authorship">2 days ago  &bull;
                                <a class="gray-link" href="/project/7274/log/23332#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-23332"><p>The refreshable braille display will be an electro-mechanical device that will display braille characters. For the moment I am using a graphical display to demonstrate the functionality of it but the future version will have a new type of display. </p><p>The upcoming braille display will consist of electromechanical character cells and each will rise or lower a combination of six round-tipped pins. The mechanism behind it uses the piezo effect of a crystal which will expand when voltage is applied thus pushing the pin up. </p><p>There has to be a crystal for each dot of the display so one character will consist of 6 crystals.</p></div>
