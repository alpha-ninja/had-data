                            <h2><a href="/project/1454/log/3513-elements-light-module">Elements - Light module</a></h2>
                            <small class="authorship">a year ago  &bull;
                                <a class="gray-link" href="/project/1454/log/3513#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-3513"><p>In this project log I'll build the whole construction from the beginning. To understand, I started to plan this project almost a year ago, when I realized the possibility to create soldering mask at home. I don't need UV for my PCB trace manufacturing because I'm using toner transfer method for it.</p><p>The construction has the following elements:</p><p>Light module: One of my major concern was, that I didn't like to etch or order a 20x30cm PCB. Etching it, drilling it is a pain. Ordering on the other has high cost. So the design is a little bit tricky. The light module become modular.&nbsp; I choose a 10x10cm size. Ordering 10pcs of 10x10cm board from seeedstudio cost me $25. It is affordable for this project and I use six of it to achieve the required 20x30cm size. I wanted to be able to control the current flow of the LEDs almost individually to be able to create equal current distribution. It lead me to not use parallel connection between the LEDs. Use certain amount of it in series, control the current flow of this serial lines and connect this regulated lines in parallel. This setup has the advantage to be able to equalize the current, and it has a two drawbacks: Relative high operating voltage and high number of control elements (darlington arrays and trimmers in this case). The board of the module design trick allows me to connect any number of modules in series. The module has three possible configuration:</p><p>Upper module:</p><p>Contains no control elements. The last line of LEDs are put in the board rotated by 90 degrees. This configuration connect the anode of the top LEDs together. This will be the high side of the voltage supply. This board has no connectors on the upper side.</p><p>Middle module (finally it is not in this configuration):</p><p>Same as the Top module, without the LED rotation and upper side connectors.</p><p>Low side module:</p><p>Same as the middle module, but with all of the control elements included, and the lower side module connectors are changed to the power and control connector.&nbsp;</p><p>Here is the schematic:</p><p><img src="https://cdn.hackaday.io/images/8268021402574540860.JPG"></p><p>The PCB design:</p><p><img src="https://cdn.hackaday.io/images/3913841402574658244.JPG"></p><p>The board back from seeedstudio:</p><p><img src="https://cdn.hackaday.io/images/1962941402575006776.JPG"></p><p>The partially populated boards (some resistors and the trimmers missing):</p><p><img src="https://cdn.hackaday.io/images/8259061402575373548.jpg"></p></div>
                            </div>

                            <h2><a href="/project/1454/log/17356-change-of-concept">Change of concept</a></h2>
                            <small class="authorship">4 months ago  &bull;
                                <a class="gray-link" href="/project/1454/log/17356#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-17356"><p>This project is still lying around in my workshop. The biggest problem I had with it, what kind of glass/plexy/etc. should I use on the top of the LEDs to be able to allow the UV light to go through it. Even I scored a UV integrator on the eBay (for ~$20) to be able to measure the performance of the LEDs through the chosen material.</p><p>On a day just pop into my mind that I was thinking in the wrong way all the time.</p><p>What if I turn the whole thing upside down?</p><p>1. I don't need any material between the LEDs and the mask.</p><p>2. When I put the mask on to the work piece I can see the alignment, not just guess it.</p><p>Now the next step is designing and building a new enclosure. Even I went further with the thinking. I put the whole thing into a flat closed box (without an lid what can be opened) and add a drawer for the work piece. This way it occupies less space, because I can put things on the top of it.</p></div>
                            </div>

                            <h2><a href="/project/1454/log/20615-display-build-success-failure">Display Build - Success? Failure?</a></h2>
                            <small class="authorship">a month ago  &bull;
                                <a class="gray-link" href="/project/1454/log/20615#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-20615"><p>I built the new display panel in the night:</p><p><img src="https://cdn.hackaday.io/images/7808781436327386036.JPG"></p><p>I tested it. Works...</p><iframe style="width: 500px; height: 281px;" width="500" height="281" src="//www.youtube.com/embed/zo42zX2cK_Y" frameborder="0" allowfullscreen=""></iframe><p>...but I can't call it a complete success. As you can('t) see the display is so dim, that it is almost invisible.</p><p>I know the reason why. It dated back in a few years. Let say five years ago, I wanted to build something. I bought a LED display and some drivers to it. The build didn't happened. To be honest I don't even remember, what was it. The LED display left in my drawer.</p><p>When I started to build this lightbox, I was thinking, that is a good idea to use the display from the drawer. It is a HDSP-B09G display from Avago (formely Agilent (formely HP)). As I figured out this is a real old design, with low intensity LEDs.</p><p>Yesterday when I realized, this is a crap, I started to search something with the same pin configuration. The result is NULL - except Avago's minimally newer HDSP-B03E. I'm not confident, that this one is really brighter.</p><p>So I've two choices:</p><p>1. Try to buy and use the "brighter" Avago</p><p>2. Redesign the display once more, and create a new board (Nooooooooooooo!!!!!)</p><p>So I order the Avago display, and we will see.</p></div>
