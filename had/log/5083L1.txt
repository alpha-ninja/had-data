                            <h2><a href="/project/5083/log/16757-tyos-024">TYOS 0.2.4</a></h2>
                            <small class="authorship">4 months ago  &bull;
                                <a class="gray-link" href="/project/5083/log/16757#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-16757"><p>I have been working on TYOS over the past week and have fixed many bugs and added a couple of new features. </p>Bugs Fixed:<ul><li>Long distance calling now accepted by call app</li><li>Apps automatically exit when phone detects incoming call</li><li>Messages are no longer accidentally considered calls</li><li>Battery percentage properly extracted</li><li>Overall cleanup of code</li></ul>New Features:<ul><li>New Power app! Instead of instant shutdown when power app is opened, a small menu with options to shutdown, logout, or cancel appears. Logout is great for developers.<img src="https://cdn.hackaday.io/images/9351091429566141264.JPG"></li><li>Reception of sms messages! Before you could only send sms messages, but now receiving them is possible thanks to the redesigned messaging app.<img src="https://cdn.hackaday.io/images/1528471429566167776.JPG"></li><li>Contacts! Incoming messages will now associate names with numbers. Contacts are handled in the <code>/home/pi/tyos/configure/contacts.conf</code> Create the directory and file and then add a line with this format: <code>Name=number</code></li><li>Low battery warning! When your phone's battery drops below 30% a warning appears on the main screen.<img src="https://cdn.hackaday.io/images/691611429567295678.JPG"></li><li><code>--version</code> tag. Use this when running <a href="http://main.py" target="_blank">http://main.py</a> to get your version of TYOS. Automatic update checking coming soon</li></ul><p><span></span><strong>TYOS is still under heavy development so check back soon for updates!</strong></p></div>
                            </div>
                        </li>
                    
                        <li>
                            <h2><a href="/project/5083/log/19356-tyos-049">TYOS 0.4.9</a></h2>
                            <small class="authorship">2 months ago  &bull;
                                <a class="gray-link" href="/project/5083/log/19356#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-19356"><p>A couple days ago I released <a href="https://github.com/spadgenske/TYOS/releases/tag/0.4.9" target="_blank">TYOS 0.4.9</a>, below is a list and some pictures of the new features. Note that some of the features were actually released in TYOS 0.3.2.</p><h2>New Features</h2><ul><li>Error logs. When an error occurs it is stored in <code>tyos/logs/tyos.log</code></li><li>Audio config file. Switch your audio mode from built in (1) to external (0) in the config file<code>tyos/configure/audio.conf</code></li><li>Added debug mode to framebuffer.</li><li>More than four apps can be installed. New app list</li></ul><p><img src="https://cdn.hackaday.io/images/2769601434055233353.JPG"></p><ul><li>Settings app. Delete SMS messages and toggle on and off the headset for phones without built in speaker and microphone.</li></ul><p><img src="https://cdn.hackaday.io/images/9894551434055379792.JPG"></p><p>Along with that, over 15 bugs were squashed with some help from Todd Riesz. Thanks Todd!</p><h2>How to Update</h2><ol><li>Remove old TYOS build with <code>sudo rm -rf tyos</code></li><li>Download new edition with <code>wget <a href="https://github.com/spadgenske/tyos/archive/0.4.9.zip">https://github.com/spadgenske/tyos/archive/0.4.9.zip</a></code></li><li>Unzip package with <code>unzip 0.4.9</code></li><li>Rename with <code>mv TYOS-0.4.9 tyos</code></li></ol></div>
                            </div>
                        </li>

                            <h2><a href="/project/5083/log/23085-tyos-051">TYOS 0.5.1 </a></h2>
                            <small class="authorship">3 days ago  &bull;
                                <a class="gray-link" href="/project/5083/log/23085#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-23085"><p>In previous releases of TYOS, the software depended on <a href="https://github.com/adafruit/adafruit-pi-cam">adafruit-pi-cam</a>, a point and shoot camera project from Adafruit. As you can see in the <a href="https://youtu.be/H2AY7ciEJvo?t=1m40s">original demonstration video</a>, the software is slow, pictures are low resolution, and only half of the display is used. All that has changed in the new release. </p><p><img src="https://cdn.hackaday.io/images/99691439748844116.JPG"></p><p><img src="https://cdn.hackaday.io/images/9605271439749005962.JPG"></p><p>The new camera app takes full advantage of the entire screen, loads pictures almost instantly, and instead of an endless &quot;Loading...&quot; prompt, a subprocess is created to handle the uploading to Dropbox to reduce waiting times by over half.</p></div>
                            </div>
