Firstly i have decided to get rid of the soil hydrometers until i can use something that measures them more reliably as the probes I'm using are corroding very quickly which is changing the readings dramatically, and really don't want to flood the plants. The four button menu i have created is very simple as i have just assigned a 4 pins as pull-ups and to be a LOW INPUT. When either &nbsp;buttons 3 or 4 are pressed this runs, the pump will run until i release my finger.</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;//ACTIVATE PUMP 1</p><p>  if (digitalRead(but3) == LOW)</p><p>   {digitalWrite(pump1, HIGH);</p><p>   lcd.clear();</p><p>   lcd.setCursor(2,1);</p><p>   lcd.print("WATERING POT ONE");</p><p>Serial.println("WATERING POT ONE");</p><p>    Serial.println();}</p><p> else</p><p>   {digitalWrite(pump1, LOW);}</p><p>Buttons 1 and 2 are going to be used for displaying the states of the sensors and outputs. We will see how this idea plans out.</p><p>Iv added a RTC (Real Time Clock) to control the lighting, I am using the&nbsp;virtuabotixRTC&nbsp;library to take care of this my light timer is taken care of with&nbsp;another if statement,</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;// LIGHT OUTPUT TIMER CONTROLL</p><p>         if (((myRTC.hours) &gt;= 8) &amp;&amp; ((myRTC.hours) &lt; 20))</p><p>         {digitalWrite(light, HIGH);}</p><p>else</p><p>        {digitalWrite(light, LOW);}</p><p>Its a&nbsp;very&nbsp;simple timer but it seems to work quite well.</p><p>Im thinking about using something similar to control the water pumps maybe on a daily or every odd day will play about with this when i get the system running with a plants.</p><p>The pic below is the project in its current state controlling a light.</p><p><img src="https://cdn.hackaday.io/images/5744191405197676626.JPG"></p><p>Here is a pic of the system manual watering, The blue LED above my finger is the OUTPUT.</p><p><img src="https://cdn.hackaday.io/images/7981221405197783940.JPG"></p><p>Here is a demo video of the timer in action.&nbsp;</p><p><iframe allowfullscreen width="500" height="281" src="//www.youtube.com/embed/5u1PoAXJiw0" frameborder="0"></iframe></p></div>

                            <small class="authorship">9 months ago  &bull;
                                <a class="gray-link" href="/project/1476/log/11143#discussion-list">

                            <div style="position:relative">
                                <div id="post-body-11143"><p>Well its been a while since i last updated my progress and as you can see I've had a lot of progress. I've finally got round to finding a suitable enclosure to house everything, Im just waiting for some 3 pin plug sockets. I now have my soil sensors with custom probes installed and working so rather than just watering the plants daily they will only get watered when needed. The soil gets checked every hour and if soil is above a set level then will activate the water pumps for user set duration (can be set from front panel :) ). All fans and light are now running off 230v and controlled via individual 40A solid state relays. I am in the process for adding a heater as I've noticed the temperature dropping an the night. </p><p>Heres a pic of the insides a little bit messy, at the bottom right of the box i have installed 5x 3.5mm stereo jacks and made the two Yellow jacks for temperature sensors, Green is for water pumps, and Blue for the soil sensors.</p><p><img src="https://cdn.hackaday.io/images/5488401416248760792.jpg"></p><p>Heres the main control panel from the front</p><p><img src="https://cdn.hackaday.io/images/2024091416248628615.jpg"></p><p>3.5mm jacks just installed.</p><p><img src="https://cdn.hackaday.io/images/5092741416249019648.jpg"></p><p>the 3.5mm jack panel, I originally only was using it for inputs then later added the centre jack for the water pumps.</p><p><img src="https://cdn.hackaday.io/images/585961416249075464.jpg"></p><p>A close up of the soil probes.</p><p><img src="https://cdn.hackaday.io/images/9975371416249169467.jpg"></p></div>
