                            <div style="position:relative">
                                <div id="post-body-21704"><p>So, I&apos;m about half way through my schematic, it&apos;s tough with the amount of free time I get but it&apos;s coming along.<br></p><p>Giving Altium CircuitMaker a go, normally I&apos;d use Eagle but I figure I&apos;m so far out of practice it doesn&apos;t make much difference either way.</p><p>Have decided to compromise, in order to be lazy and keep using ATmega32u4 - Mostly for the simplicity of built in USB - frame buffering will be done on the host and streamed via the MCU to the EPD.</p><p>I could render the frame for a 2 inch EPD on the MCU but it uses 2.4k out of the 2.5k of memory so it&apos;s just going to be easier this way - it&apos;ll be faster than relying on external flash and a lot simpler than picking a bigger MCU and working out USB.</p></div>
