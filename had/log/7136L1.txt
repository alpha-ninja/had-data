The Extension port for this version has 4 I/Os and JTAG pins, 1.8V logic level. There is one LED, as sensor system Si1143 is used with one IR LED.</p><p><img src="https://cdn.hackaday.io/images/6250471439145072030.png"></p><p>Geophyte conversion to Vivado is WIP, it seems that linux machine with python &gt;= 2.7 is required to generate the wishbone system. The SD Card part of geophyte did synthesis OK with Vivado 2015.2 funny google guys still use ISE 14.7 !</p></div>
