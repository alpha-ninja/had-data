                            <h2><a href="/project/4910/log/16402-sensing-heart-beat">Sensing heart beat</a></h2>
                            <small class="authorship">4 months ago  &bull;
                                <a class="gray-link" href="/project/4910/log/16402#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-16402"><p>So far I've got the piezo sensor picking up my heart beat on my wrist, The heart rate is displayed on a 1602 LCD screen. For better visuals from a distance I added a red LED and a piezo speaker. I currently received a DS18B20 temperature sensor which looked great and came in the waterproof enclusore. But after added the code and library to my project I soon realized that it was too slow and messed up my timing for my piezo vibration sensor. My plan to over come this is to put a LM35 into the DS18B20 steal enclosure.</p></div>
                            </div>

                        <li>
                            <h2><a href="/project/4910/log/17045-a-large-enclosure-requires-a-large-pcb">A large enclosure requires a large PCB</a></h2>
                            <small class="authorship">4 months ago  &bull;
                                <a class="gray-link" href="/project/4910/log/17045#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-17045"><p>I swapped out the DS18B20 temperature sensor since it was messing up my timing in my sketch and replaced it with LM35 and its been working great. </p><p>Since the temperature sensor was the last bit of hardware I thought I should start to pick out an enclosure. With every new project I like to try new/different things so I picked a large desktop enclosure. </p><p><span></span></p><p><img src="https://cdn.hackaday.io/images/1512961430162824808.jpg"></p><p>One that would have plenty of room for all of the components to fit into. The thing about a big enclosure is that it will require a larger then I would normally be able to afford PCB. I really do hope I win some OSHPARK vouches so I can order my dream PCB for this and other projects and not have to worry about size</p><a href="http://www.ebay.co.uk/itm/390924433631?_trksid=p2057872.m2749.l2649&amp;ssPageName=STRK%3AMEBIDX%3AIT" class="vip item-title"></a></div>
                            </div>
                        </li>
                    
                        <li>
                            <h2><a href="/project/4910/log/18669-pcb-and-enclosure">PCB and enclosure</a></h2>
                            <small class="authorship">3 months ago  &bull;
                                <a class="gray-link" href="/project/4910/log/18669#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-18669"><p>So finally my PCB turned up from <a href="http://oshpark.com">oshpark</a> and it quickly dawned on my that some of the drill holes looked on the small side. On closer inpection it was confirmed that they were too small. This wasnt really a problem because the drill holes were big enought to solder some wire though so i would just have to break them out that way. </p><p><img src="https://cdn.hackaday.io/images/6499651432916654414.jpg"></p><p>It only really affected the FTDI connector and the speaker too.</p><p>Now that I had all my parts I could start thinking about how they were going to go into the enclosure. Started with the hardest part by cutting the hole for the LCD screen. I love this enclosure because it all comes apart so its easy to grip it in the vice ect. </p><p><img src="https://cdn.hackaday.io/images/4007911432916951637.jpg"></p>Next to the LCD screen is the two 3.5 mm head phone jacks and two buttons. Under the screen lives the LED for the heartbeat.<p><img src="https://cdn.hackaday.io/images/5674231432917043843.jpg">Round the back there is the single throw double pole switch that handles the power on and the battery monitor wire. Also there is a usb B for charging and data upload. Above the usb b port there is a small switch so you can select between charging and just power<br></p></div>
                            </div>
                        </li>

                        <li>
                            <h2><a href="/project/4910/log/20884-powered-by-light">Powered by light</a></h2>
                            <small class="authorship">a month ago  &bull;
                                <a class="gray-link" href="/project/4910/log/20884#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-20884"><p>When I started this project I always wanted it to be able to be powered by solar. The only experience I&apos;ve had with solar cells is messing about with the garden solar lights but they dont produce much voltage/current.</p><p>I started looking around on ebay to see if I could find a 5v panel with decent output. I picked a 5v 500mAh 2.5 watt panel hoping that it would be able to power <a data-mention-project-id="4910" href="/project/4910">#T_H_S</a> . Being in the UK we dont normaly get many sunny days so on recieving the panel I had to wait a few days before testing the panel out.</p><p><img src="https://cdn.hackaday.io/images/683201436824876795.jpg" width="452" height="255" style="width: 452px; height: 255px;"></p><p>I could get 5.90v with strong direct sunlight. Now I needed a way to boost/regulate the current so it would power <a data-mention-project-id="4910" href="/project/4910">#T_H_S</a> . </p><p>I ordered a 5v DC - DC boost</p><p><img src="https://cdn.hackaday.io/images/9449211436824597346.jpg" width="391" height="286" style="width: 391px; height: 286px;"></p><p>This allowed me to power <a data-mention-project-id="4910" href="/project/4910">#T_H_S</a> as long as it was in strong direct sunlight. I would charge and power the battery&apos;s at the same time but now too sure how long it would take to fully charge them up.</p><p><img src="https://cdn.hackaday.io/images/9875921436824842328.jpg" width="424" height="237" style="width: 424px; height: 237px;"></p><p>I mounted the solar cell on a piece of PVC ment for a window ledge so it was L shaped. This alowed it to have a slight angle to chatch the sun a little better</p><p>Next thing I&apos;m waiting on is a hand crank phone charger.</p></div>
                            </div>
