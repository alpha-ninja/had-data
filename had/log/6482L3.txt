                            <h2><a href="/project/6482/log/22728-system-design-document">System Design Document</a></h2>
                            <small class="authorship">5 days ago  &bull;
                                <a class="gray-link" href="/project/6482/log/22728#discussion-list">
                                    0 comments
                                </a>

                            <div style="position:relative">
                                <div id="post-body-22728"><p>This System Design Document shows the hardware components of this project, the different screens of the Arduino Enigma Simulator, how to operate it and encrypt your first message.</p><p>The two main components in this build are the Seed Studios TFT Touch Shield and an Arduino Uno R3. A 9V Battery and a barrel plug cable supply power to the project. </p><p>The real machine had a wood case, a laser cut plywood enclosure was designed for this project. The original logo is laser engraved inside the lid and a paper tag with operation instructions and the machine serial number was recreated to the appropriate scale and glued to the inside of the lid.</p><p>The enigma algorithm has been thoroughly verified for correctness since v1.0. All know glitches in user interface have been corrected as of v1.4. Further improvements include flipping the image vertically so that the power and USB connectors are in the back of the machine. The image of a plugboard would be then laser engraved on the side of the case. </p><p></p><p>These are the two main components of this project. Using of-the-shelf unmodified parts was a design choice to allow easy mass production.</p><p><img src="https://cdn.hackaday.io/images/3495661439762190824.jpg"></p><p><img src="https://cdn.hackaday.io/images/5052131439762225082.jpg"></p><p><img src="https://cdn.hackaday.io/images/7882631439762241757.jpg"></p><p>When plugged together and loaded with the enigma program, it looks like this:</p><p><img src="https://cdn.hackaday.io/images/6273901439762286937.jpg"></p><p>encased looks like this:</p><p><img src="https://cdn.hackaday.io/images/5102481439762377961.jpg"></p><p>The enigma machine is operated and configured from three different screens. In the home screen, the wheels can be set, the keys pressed, and the lamps illuminate. From that screen, the user can go the the plugboard view by touching the plug area at the bottom of the home screen or configure the wheels by touching the enigma logo in the home screen. From the open machine view, the user can go to the plugboard view by touching the area at the bottom of the screen.  Once in the plugboard view, the user returns to the previous screen by touching the keys at the top of the screen.</p><p>The about screen is shown upon the first 3 times power is applied.<br><br>This screen clarifies that this device is for educational, historical and entertainment purposes. The encryption principle implemented has been broken many times in the past.<br><br>A list of online resources that provided information used to build this implementation are listed.<br><br> Touch OK to continue to the main screen.<span class="redactor-invisible-space"><br></span></p><p><img src="http://4.bp.blogspot.com/-5A5Hp3InPgo/VEcnYg3o3WI/AAAAAAAAABM/WDZlPIbvSp8/s1600/1.jpg"></p><p>This is the home screen, the Enigma machine is ready to encode messages with the settings in effect at the moment. If this is the first time the machine is powered up, the default settings are loaded, otherwise the last settings used are kept.<br><br>To start encoding a message, press and hold the keys at the bottom of the screen, the wheels will move, a lamp in the middle section will illuminate for as long as the key is pressed and the corresponding character will also be printed in the tape running above the lamp field. Once the key is no longer pressed, the lamp will turn off.<br><br>The default setting of AAAA can be changed by dragging the rings to the right of the corresponding letter up or down.<br><br>Encoding a message with the default settings is no fun, so let&apos;s touch the Enigma logo to open the machine and be able to change the rotors. <br></p><p><img src="http://1.bp.blogspot.com/-0G1hLnS6qKc/VEcnZimV77I/AAAAAAAAABo/0q0szkEm194/s1600/2.jpg">The machine is now open, notice that the bulbs in the lamp field are exposed, the rotors are shown, the printer, texture, serial port functions can be changed and a box of spare rotors is shown at the bottom of the screen.<br><br>To start changing the rotors, the ones installed must first be removed, to do so, touch the letters in the rotors to be removed.<span class="redactor-invisible-space"><br></span></p><p><span class="redactor-invisible-space"><span class="redactor-invisible-space"><img src="http://1.bp.blogspot.com/-coja8EnfxFs/VEcncNCvcEI/AAAAAAAAACs/EY9D6X4QWPg/s1600/3.jpg">Rotor I has been brought down to the middle area and placed flat on its side.
