                    <h5>Description</h5>
                    <div class="description">This is a project idea. This is the way I work. Millions of things come in my mind, some of it goes nowhere, some of it never finished and some of it grows to a finished project/product.<br /><br />If you measure the voltage of a USB connector of a PC it would never be 5V. In a good case 4.5-4.7V but it can be worse. Some of the equipment can't live with it (for me the STM32F429 Discovery board was the worst experience).<br />The goal of this project to create a small device what can be connected into the USB line, contains a small boost regulator and create precise 5V for the picky equipments I have.</div>

                        <li>
                            <h2><a href="/project/2255/log/20240-faliure-no-2">Faliure no 2.</a></h2>
                            <small class="authorship">2 months ago  &bull;
                            <a class="gray-link" href="/project/2255/log/20240#discussion-list">
                                11 comments
                            </a>
                            </small>
                            <p><p>Created a second board, now with the corrected USB connector pinout:</p><p><a href="http://4.bp.blogspot.com/-68ufUr5HaD4/VZKOotjvxbI/AAAAAAAAD-k/Ss-17mHI0r4/s1600/USB_Booster_Pcb_v2.JPG"><img src="https://4.bp.blogspot.com/-68ufUr5HaD4/VZKOotjvxbI/AAAAAAAAD-k/Ss-17mHI0r4/s400/USB_Booster_Pcb_v2.JPG"></a></p><p>And I built it:</p><p><a href="http://2.bp.blogspot.com/-YWCTdYNNsQQ/VZKPH4_D3CI/AAAAAAAAD-s/EExstLDD_sY/s1600/USB_Booster_V2_build.JPG"><img src="https://2.bp.blogspot.com/-YWCTdYNNsQQ/VZKPH4_D3CI/AAAAAAAAD-s/EExstLDD_sY/s400/USB_Booster_V2_build.JPG"></a></p><p>The problems started here.</p><p>The first thing, I wanted to test it. To test it it is not enough to connect it to some random USB source. I wanted a variable source.</p><p>I've a lab supply, but it has no USB out. As I didn't wanted to cut any USB cables, I designed and built this small "tool":</p><p><a href="http://1.bp.blogspot.com/-pOLp236hMuI/VZKSBqPBS_I/AAAAAAAAD-4/upSJNH0Lay8/s1600/Lab%2BSupply%2BUSB%2BBreakout.jpg"><img src="https://1.bp.blogspot.com/-pOLp236hMuI/VZKSBqPBS_I/AAAAAAAAD-4/upSJNH0Lay8/s400/Lab%2BSupply%2BUSB%2BBreakout.jpg"></a></p><p>After this the testing went to a complete disaster.</p><p>1. It was working from ~2V to 4.3V correctly, but when I went above 4.3V the output voltage just went above 5V. I was thinking this can be the result of miss of the minimal load, what usually required. So I setup the circuit with some (~50mA) load:</p><p><a href="http://3.bp.blogspot.com/-DfD9ES4ctEA/VZKa2gi0twI/AAAAAAAAD_I/hAzLZxsr4qY/s1600/USB_Booster2_test.jpg"><img src="https://3.bp.blogspot.com/-DfD9ES4ctEA/VZKa2gi0twI/AAAAAAAAD_I/hAzLZxsr4qY/s400/USB_Booster2_test.jpg"></a></p><p>Testing..</p><p>At 2.0V:</p><p><a href="http://2.bp.blogspot.com/-bVgUTC_EfaM/VZKbDzewCTI/AAAAAAAAD_Q/P6QNLyYOL84/s1600/USB_Booster2_test_2V.jpg"><img src="https://2.bp.blogspot.com/-bVgUTC_EfaM/VZKbDzewCTI/AAAAAAAAD_Q/P6QNLyYOL84/s400/USB_Booster2_test_2V.jpg"></a></p><p>Looks good</p><p>At 4.3V:</p><p><a href="http://3.bp.blogspot.com/-20SrEg7O7gM/VZKbQTOHRAI/AAAAAAAAD_Y/H8xwpbflINQ/s1600/USB_Booster2_test_4.3V.jpg"><img src="https://3.bp.blogspot.com/-20SrEg7O7gM/VZKbQTOHRAI/AAAAAAAAD_Y/H8xwpbflINQ/s400/USB_Booster2_test_4.3V.jpg"></a></p><p>Still looks good</p><p>At 4.4V:</p><p><a href="http://1.bp.blogspot.com/-2IPHJW2HJ1w/VZKbeJM-XNI/AAAAAAAAD_g/8n0fYAnSnzE/s1600/USB_Booster2_test_4.4V.jpg"><img src="https://1.bp.blogspot.com/-2IPHJW2HJ1w/VZKbeJM-XNI/AAAAAAAAD_g/8n0fYAnSnzE/s400/USB_Booster2_test_4.4V.jpg"></a></p><p>Not so good anymore. :-(</p><p>In addition, if I change the voltage, sometimes the whole circuit gets unstable. So the result clearly useless.</p><p>I've possible explanations to the things above:</p><p>- I've a crap layout - I'll try to enhance it in the next version</p><p>- I've some sort circuit(-is) things in my build. I can admit that using solid groundplane around the surface mount inductors without insulating solder mask wasn't my best idea</p><p>- The boost regulator is unable to handle the &gt;4.3V in 5V out situation even if the datasheet just said Vout&gt;Vin as a requirement</p><p>Here are the possible solution for the next try:</p><p>- Create a new board, with different layout and definitely with a cutout in the groundplane around the inductors</p><p>- Use the pass trough capability of the chip (this will require some kind of external comparator)</p><p>- Use a different (buck-boost or SEPIC) circuit - I may try the LM2621 what I ordered already for one of my other projects</p><p>- Keep the current boost converter (rising the output voltage to - let say - 6 volts) and use it as a pre-regulator for an LDO. As It suggested by <a href="https://hackaday.io/TheBigOne" target="_blank">The Big One</a>.</p>
                                
                            </p>
