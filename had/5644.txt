                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">The average lawn uses tens of thousands of gallons of water per year. This water isn't unlimited. But is all of it necessary? Consider, for example, the automatic sprinkler system that waters during a rain storm. The motivation for this project was to save water, plain and simple.<br /><br />The On-Demand Sprinkler System is designed to measure soil moisture, interface with existing irrigation systems, and determine whether or not the user's lawn should be watered.<br /><br />The sensor units feature custom-engineered soil moisture sensors, battery power, solar panels, and wireless communication. The main unit utilizes wireless communication to talk to the sensor units and receive the soil moisture measurements.<br /><br />This was a Senior Design project during the 2014-15 school year at Oregon State University. The team consisted of Chelsea Collette, Alex Crum, and Tim Blikstad. The project was successful in all its objectives, but future improvement would add more features and make sturdier sensor units.</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>All design details, including components used, design calculations and decisions, CAD drawings, and PCB files for this project can be found on the Senior Design website, which is linked on this page. The following is a summary:</p><p>The moisture sensors utilize a capacitor built out of two aluminum rods. The sensor measures the electrical properties of the soil between the rods, and from this it determines the moisture of the soil. Basically, when there is more water, the rods look like a bigger capacitor. The accuracy of this sensing technique can be calibrated to be within 10% of the measured value.</p><p>Wireless communication is achieved through a mesh network of Zigbee devices. This means that a sensor placed far from the house could communicate with the main unit through another sensor. This allows for great range, as well as allocating a sensor for each sprinkler zone.</p><p>The main unit determines whether or not to allow irrigation based on two things: a user-specified moisture threshold, and the data from the sensors. If a zone is too dry, the attached sprinkler system is allowed to irrigate. If a zone is wet enough, the system halts irrigation to save water.</p>
                                
                            </p>
                        </div>
