                    <h5>Description</h5>
                    <div class="description">Commercial ground penetrating radar systems cost thousands of dollars. This project aims to create an open hardware alternative for about $500, using a Raspberry Pi with PiMSO to save data for later analysis and send instantaneous results to a web browser on a tablet or smartphone.<br /><br />Why  this project matters: SAFETY!<br /><br />Eight people died in the 2010 San Bruno pipeline explosion.<br />May residents didn't even know there was a gas pipeline in the neighborhood.<br /><a href='http://en.wikipedia.org/wiki/2010_San_Bruno_pipeline_explosion' target="_blank">http://en.wikipedia.org/wiki/2010_San_Bruno_pipeline_explosion</a><br /><br />The Gates of Hell Opened In Guatemala, also in 2010:<br /><a href='http://gizmodo.com/5551916/the-gates-of-hell-just-opened-in-guatemala' target="_blank">http://gizmodo.com/5551916/the-gates-of-hell-just-opened-in-guatemala</a><br /><br />Many towns in Illinois are built over abandoned mines.<br />It is only a matter of time before some of them cave in.<br /><br />This project changes the world by enabling people to discover these hazrds for themselves. It will leverage OpenStreetMap to share these dangers with communities, so that they can be mitigated before a disaster.</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>A Baofeng VHF/UHF radio is used a signal generator, which transmits via a Transmit/Recieve switch through an antenna. A  RF detector is also connected to the T/R switch, which is connected to the MSO-19.</p><p>The radio controller is a 4N25 opto-isolator.</p><p>The T/R switch controller is a SN74HC04N (or similar) hex inverter.</p><p>Any Pi-compatible GPS and WiFi dongle will do.</p><p>The Raspberry Pi triggers the radio, T/R switch and MSO-19 via a buffered GPIO port once per second using a very simple python script.</p><p>In the United States, this project falls under FCC Part 97 <span class="subheads">Sec. 1 (b)</span>:<br><em>"Continuation and extension of the amateur's proven ability to contribute to the advancement of the radio art."</em><br></p>
                                
                            </p>
