                    <h5>Description</h5>
                    <div class="description">I recently bought a rather cheap Quadcopter (called "copter" from now on for convenience) from a Toystore - the Salesman advertised this one greatly. For about 99€ (~ 110$) there was no long thinkering about getting this Drone...<br /><br />I got this one (Sorry only german): <br /><a href='http://monstertronic.org/Website/index.php/multicopter/modelle/scopter-ufo-mit-kamera-und-wechselmodulen-optional' target="_blank">http://monstertronic.org/Website/index.php/multicopter/modelle/scopter-ufo-mit-kamera-und-wechselmodulen-optional</a><br /><br />The nice thing is, that it is prepared to attach different Modules - so it has to be powerful enough to lift some additional load.</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>In my case i want to add an @atmega Heart to the copter. As first step i want to design a "Module" which attaches and detaches to the copter like the stock ones. </p><p>This project includes:<br>- Designing a 3D-Printable Bracket for holding the PCB (f.Ex. the <a data-mention-project-id="1853" href="/project/1853">#The Atmega Hearth</a> and a matching LiPo<br>- Searching for Plugs which would be matching into the holes of the copter "mainboard"<br>- The module feature an attached RGB LED for visibility and awesomeness / Lightpainting<br>- Add some fancy logging Stuff<br>- Inspire others</p>
                                
                            </p>
                        </div>
