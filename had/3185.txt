                    <h5>Description</h5>
                    <div class="description">The Dynamic Input / Output module is a piece of hardware, a USB computer peripheral, that allows users to quickly and easily connect any computer with the physical world in a way that has never been easier. Plug sensors, LEDs, servos and other peripherals directly into the DyIO’s channels and then use a normal computer or laptop to get values from sensors and set values for outputs.<br /><br /></div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p><strong>Easy to Use</strong></p><p>Connect the DyIO with a USB cable and start playing with sensors, motors and other pieces in minutes You can start collecting data, moving servos and setting outputs without writing a single line of code. Coding with the DyIO couldn&apos;t be easier. Check out the &apos;No Programming&apos; development application <a href="https://hackaday.io/project/6423-bowlerstudio-a-robotics-development-platform" target="_blank">BowlerStudio</a>. Once you&apos;re ready to begin programming, check out our <a href="http://neuronrobotics.github.io/Java-Code-Library/Digital-Input-Example-Simple/" target="_blank">programming tutorials</a> to see some examples.</p><h2>Bowler Studio</h2><p>BowlerStudio is a robot development application that combines scripting and device management with powerful control and processing features.</p><p>BowlerStudio brings the full power of The OpenCV image processing library, a Configurable kinematics engine based on D-H parameters, A Fully Featured JCSG based Cad and 3d Modeling engine, and a Built in WebKit browser capable of viewing documentation and executing code directly from Github Gists.</p><p>Scripts can be loaded from the Local File System and edited using a editor of your choice or loaded from Github gists for a truly &quot;backed up in the cloud&quot; coding experience.</p><p>BowlerStudio features the Creature Creator, a design interface that lets users generate robots with slide bars. The application will generate 3d printable STL&apos;s that will walk just like the generated robot on the screen.</p><strong>Cross Platform</strong><p>We&apos;ve tested the DyIO on all major operating systems. This means that users are able to write code on Mac deploy to Linux and test in Windows without having to worry about different tool chains. Users are able to get up and running in as little as 8 minutes. Likewise, you can use almost any programming language or environment, currently implemented in Java, Python (Jython). Any other language can be added as well, from C to MatLab to Excel! Using the Neuron Robotics Development Kit, users already have a powerful library out-of-the-box with standard Java SE.</p><p><strong>GitHub Gist Collaborative Robot Coding</strong></p><p>We have developed a unique application with BowlerStudio. The application is basically a web browser with the ability to run scripts right out of GitHub Gist. You just browse to a Gist, or to a web page with an embedded gist, and hit run. It loads up the code and can run a robot, build CSG style CAD objects, and talk to lots of other hardware useful for building robots. </p><strong>Dynamic Inputs &amp; Outputs</strong><p>The DyIO has 24 channels that can configure to be whatever mix of inputs and outputs is need at the time. The inputs can be set to asynchronous modes so that the DyIO will send alerts when there are changes versus having to poll inputs constantly.The outputs can use the USB line for power or can use the external battery connection for higher current applications (i.e. servos) Drive up to 24 servos simultaneously!</p><ul><li>24x Digital Input &#x2013; All channels</li><li>24x Digital Output &#x2013; All channels</li><li>24x Servo Output &#x2013; All channels</li><li>8x 10 bit Analog Input &#x2013; Channels 8 &#x2013; 15</li><li>4x PWM Output &#x2013; Channels 4 &#x2013; 7</li><li>4x DC motor controller signals &#x2013; Channels 4 &#x2013; 7, with direction flags</li><li>1x Asynchronous Serial (UART) In &amp; Output &#x2013; Channels 16 &amp; 17</li><li>1x Synchronous Serial (SPI) In &amp; Output &#x2013; Channels 0,1, &amp; 2</li><li>4x Counter/Encoder (Tick, Direction &amp; Home) Inputs</li><li>4x Stepper Controller (A, B &amp; Home toggle) Outputs</li><li>R/C PPM signal reader on Channel 23</li><li>8x Real-Time, dynamically configurable controllers (PID)</li></ul>PID in hardware<p>The DyIO features a cutting edge PID control system. This system lets you link any input to any output with the DyIO managing real-time control. Each PID group allows one input (counter or analog input) to be linked to one output (DC motor or Servo) in a real-time closed loop controller.</p>Open Communications<p>Controlled via the Bowler Communications System. Interoperates with any BCS complaint system! Addressed via IEEE standard MAC address. Built-in USB is a universal serial port. No drivers needed on most systems.
