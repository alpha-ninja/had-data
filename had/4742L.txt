                            <div style="position:relative">
                                <div id="post-body-17439"><p>Well here is a long overdue update.  So far the gas sensor ball is working extremely well in tests.  Please excuse the "ugly" prototype as I actually recycled some old parts from another project to expedite the assembly :)</p><p>Rather than typing up a long explanation- here is a full video update:<br></p><p><iframe width="500" height="281" src="//www.youtube.com/embed/THKldqk9lY8" frameborder="0" allowfullscreen=""></iframe><br></p><p>Code is now posted so all people can recreate/improve this project anywhere in the world.  I truly hope this can be used someday in real world situations.  Maybe even save a life someday...</p><p>I am not holding back on the design/code- take it, make it yours, make it better.  Share it with the world!</p><p>For now I am going on the hunt for a better sensor for this project.  the MQ2 seems to work well for a "Go/No-Go" indicator but it does not do very well for clear and specific measurements of the various levels. It does seem to become "confused" if the sensor is saturated in certain conditions.   If I can find a sensor which can be calibrated and falls within the accuracy of the commercial multi-gas analyzers I will be much happier.  </p><p>Enjoy and thanks for all the support!</p><p>Eric</p></div>

                                <a class="gray-link" href="/project/4742/log/21666#discussion-list">
                                    6 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-21666"><p>Finally got around to designing a custom PCB for the gas sensor.  I am entirely new to using Fritzing so I decided to simply use the Arduino Nano as it was without breaking out the chip, supply etc.  This isn&apos;t the best use of space but should make this super easy for anyone to replicate.  </p><p>The PCB design went really quickly and I was able to export the design files in an hour or so:</p><p><img src="https://cdn.hackaday.io/images/6188651438061392155.jpg" width="137" height="379" style="width: 137px; height: 379px;"></p><p>Off I went to the OSH Park website to use the credit I received from an early bird prize in the Hackaday 2015 contest.  The PCB&apos;s were really cheap and I ordered- but 5 minutes later I found a serious error in the design- so I re-ordered again :)</p><p><img src="https://cdn.hackaday.io/images/5516931438061509273.JPG" width="461" height="549" style="width: 461px; height: 549px;"></p>Version 2.0 looks promising and once I have verified it works I will post all the design files so everyone can create their own version. Super happy with this project and loving that it has been well received with the open source and electronics community.<p>Cheers</p><p>Eric</p></div>
