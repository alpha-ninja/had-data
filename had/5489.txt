                            <p><p><strong>1. The problem which has been chosen as the subject of the project</strong><strong></strong></p><p><strong><br></strong></p><p><span class="redactor-invisible-space">Controlling interfaces such as LCDs, push buttons, meters, etc, are a significant cost in many electronic projects. These components also take up space. Why build another physical interface when there are desktops, notebooks, tablets, smartphones, etc, ready to take the controlling interface roll?</span></p><p><span class="redactor-invisible-space"><br></span></p><p><strong>2. How this project will work to alleviate or solve the problem<span class="redactor-invisible-space"><br></span></strong></p><p><strong><span class="redactor-invisible-space"><br></span></strong></p><p><span class="redactor-invisible-space">By creating a JSON embedded function framework, functions can be defined including parameter lists and any return values. The embedded device can then be exposed via any existing transport be it wired or wireless so any paired external device is able to make remote function calls, effectively controlling the embedded device. </span></p><p><span class="redactor-invisible-space"><br></span></p><p><span class="redactor-invisible-space">As a side effect, I hope this effort will make it easier for myself to integrate multiple embedded devices.</span></p><p><span class="redactor-invisible-space"><br></span></p><p><span class="redactor-invisible-space">Fig. 1 Controlling device queries embedded device with JASONette Interface libdef keyword.
