                        <div class="post-content details-content">
                            <p><h1>Introduction</h1><p>A group of cat lovers at the University of Pretoria in South Africa run a Trap-Neuter-Return program.  This device was developed at the request of one member in order to increase the efficiency of said program and reduce cat stress and injury.</p><h2>What is Trap-Neuter-Return (TNR)?</h2><p>From Wikipedia (<a href="https://en.wikipedia.org/w/index.php?title=Trap-neuter-return&amp;oldid=674685337">link</a>):</p><p><em>&quot;Trap-neuter-return (TNR) is a program through which free-roaming cats (not belonging to particular humans) are humanely trapped; sterilized and medically treated; and returned to the outdoor locations where they were found.&quot;</em></p><p><em>&quot;Various long-term studies have shown that TNR is effective in stopping reproduction and reducing the population over time.&quot;</em></p><p>In short, TNR is a humane and effective way of controlling feral cat populations without the need to euthanise healthy cats.</p><h1>The Problem</h1><p><img src="https://cdn.hackaday.io/images/7098641439834957747.jpg" alt="" width="304" height="182" style="float: right; width: 304px; height: 182px; margin: 0px 0px 10px 10px;"></p><p>Cats need to be caught in order to be vaccinated and sterilized. This is accomplished with humane cat traps. The trap takes the form of a metal cage with a pressure plate that activates a spring loaded door.</p><p>Traditionally someone checks the traps on a daily basis and collects any trapped kitties. This is time consuming and generally limits the amount of traps that can be practically operated.  It also presents an even worse problem.</p><p>Trapped cats may become stressed and agitated and attempt to escape the trap.  This can cause injury and in some cases death.  The longer the cat remains trapped without intervention the greater the chance of a negative outcome.</p><h1>The Solution</h1><p>The device aims to solve the above problems by notifying the responsible person as soon as a trap is tripped.  This allows them to respond immediately and remove the cat before stress and injury can occur.</p><p>The device senses the trap door closing and sends a text message to a pre-programmed number.  In addition the device will respond to text messages received from said number with the sensor status and battery level.</p><p>The device consists at it&apos;s core of an Arduino Pro Mini, GSM module and LiPo battery.  A reed switch sensor is plugged in to detect the door closing.  The GSM module handles battery charging and monitoring.  A computer interface is provided in the form of a USB to Serial converter - this is used to change settings.</p><h1>The Device</h1><p><img src="https://cdn.hackaday.io/images/8413491439835426237.JPG"></p><ul><li>Power Switch: Turns unit on and off</li><li>Send Enable: Enables or disables sending of text messages (useful to turn off when testing)</li><li>Network Status: Indicates network and sensor status. Green when connected to a mobile network, red when not connected.  Flashes 3 times when sensor status changes.</li><li>Battery Status: Green when battery OK / finished charging.  Red when battery low (&lt; 20%).  Amber when battery charging.</li><li>Charger Socket: Receives 5V DC charger</li><li>Sensor Socket: Receives reed switch sensor lead</li></ul><h1>The Computer Interface</h1><img src="https://cdn.hackaday.io/images/875131439835783299.JPG"><p>The computer interface, used for programming settings, is a CR2302 breakout board.  The GND, TX and RX pins are connected to a 3.5mm stereo audio jack which is plugged into the device.</p><p><img src="https://cdn.hackaday.io/images/9386971439836031879.png"></p><p>The software pictured is used to set the notification number to send text messages to and the SIM PIN.</p><h1>The Sensor</h1><p><img src="https://cdn.hackaday.io/images/5294471439836123223.JPG"></p><p>The door sensor is a simple commercial reed switch usually used in alarm systems.  The magnet is attached to the trap door and the sensor to the trap frame.  The device reacts to any change in sensor status meaning it can be set up to trigger when the magnet moves away from the sensor or when it moves next to it.</p><h1>Operation</h1><h2>First Time Operation</h2><ol><li>A SIM card is inserted into the GSM module</li><li>The device is powered up</li><li>The computer interface is connected</li><li>The computer interface software is run</li><li>The notification number (where to send text messages) and SIM PIN (if required) are entered into the software and the &quot;Write Settings&quot; button clicked</li><li>The software confirms the settings have been written</li><li>Continue with Standard Operation</li></ol><h2>Standard Operation</h2><ol><li>The device is powered up</li><li>Wait for the &quot;Network Status&quot; light to turn green indicating a network connection</li><li>The &quot;Send Enable&quot; switch to switched to 0 (off)</li><li>The sensor is affixed to the trap frame and the...</li></ol>
