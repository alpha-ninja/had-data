                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">I wanted more functionallity in my hobbyking 6ch transmitter, so I coded a new transmitter module, for connect via PPM to the RF 2.4Ghz board. Its working grear! 5 channel memory, servo reversal, dual rates, stick calibration, and some new functions on the way. The first version uses a 16x2 LCD shield, but the new one uses a nokia 5100 graphical lcd </div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>The main idea was to replace the original board with one with more functions. The arduino reads the analog sticks from the controller, and generates a PPM signal that is sended to the frsky 2.4Ghz module that original comes with the radio.</p><p>On the first version (&nbsp;http://patolin.com/blog/2013/05/05/transmisor-de-2-4ghz-con-arduino/ ) I was using a simple GUI with a 16x2 LCD, and works great. Right now Im working with a nokia 5100 LCD, that shows more information on screen</p>
                                
                            </p>
                        </div>
