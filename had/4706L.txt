                            <h2><a href="/project/4706/log/17698-uvih-uv-index-hour">UVIh - UV Index hour</a></h2>
                            <small class="authorship">3 months ago  &bull;
                                <a class="gray-link" href="/project/4706/log/17698#discussion-list">
                                    0 comments
                                </a>
                                </small>
                            <div style="position:relative">
                                <div id="post-body-17698"><p>Sunburn can be controlled by observing the accumulated UV exposure. This measure can be described with the "UV index hour" unit.</p><p><img src="https://cdn.hackaday.io/images/2083501431326939530.png"></p><p>Some explanation for the MED - UVIh conversion:</p><p>The general MED (minimal erytherma dose) values for the different skin types (Fitzpatric skin type scale) are the following:</p><p>Type I - 200 J / m2<br>Type II - 250 J / m2<br>Type III - 350 J / m2<br>Type IV - 450 J / m2<br>Type V - 600 J / m2<br>Type VI - 1000 J / m2</p><p>Conversion to UVIh:<br>J / m2 = Ws / m2 <br>1 UVI = 25 mW / m2 <br>1 UVIh = 25 mW / m2 * 3600s = 90 Ws / m2<br>UVIh = MED / 90</p></div>
                            </div>
