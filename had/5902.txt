                    <h5>Description</h5>
                    <div class="description">Everyone knows smart homes are coming. This will be my take on it. I want to be able to save on energy costs. Monitoring the power and water usage of the home will hep with this. Also it will double as a home security system etc. Done right I will be able to tell a window is open, or if a pipe burst. Remotely monitor temperatures and adjust them according to what i want. In the long run this project is to use less energy and be more climate friendly. <br /><br />Each sub-controller will use Adafruit 2.8 tft touchscreen with an ardiuno uno for now.  The size format will allow it to fit in a standard outlet box.<br /><br />The sub-controller will control the following in each room:<br />HVAC for fan, heating and cooling.<br />Lighting of the room<br />Intercom<br />Home stereo system<br />Display Alerts<br />Alarm clock functions for the room.<br /><br />The master controller will all monitoring off all rooms and override controls in the room . Is the gateway to the internet for the system and will have timers via a calendar. <br /></div>
                </div>
