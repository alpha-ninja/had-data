                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">This is a hardware realization of Eric Smith's Nonpareil microcode simulator.<br /><br />The objective of the project is to re-create popular 70's RPN calculators such as the HP Woodstock and Spice series.<br /><br />Another goal of this project is to have the design as simple as possible so that the project can be re-produced by the average electronic hobbyist.<br /><br />A low power MSP430G2553 is used in this project. The DIP profile allows easy soldering. Only through-hole components are used. LED multiplexing is done w/ a 74hc164 shift register. LED modules are direct driven w/ pulsing to eliminate the need of current drivers.<br /><br />LED modules are used to mimic the original 70's calculators and gives a retro look to the end product.<br /></div>
                </div>

                            <p><iframe style="width: 500px; height: 281px;" width="500" height="281" src="//www.youtube.com/embed/WwPM0slZro4" frameborder="0" allowfullscreen=""></iframe><p>Parts list</p><p>__________</p><p>* msp430g2553 (or other G series dip 20pin devices w/ 16k flash)</p><p>* 74hc164 shift register</p><p>* 3x sparkfun 4 digit bubble led module</p><p>* 32x tactile buttons</p><p>* 1x CR2032 SMD battery holder</p><p>* 1x 47k, 1x 1k resistors</p><p>* 2x 100nF (104), 1x 1nF (102) capacitor</p><p>Application Notes</p><p>_________________</p><p>* press-n-hold 1st row, 5th column (from top) for hp33C</p><p>* press-n-hold 2nd row, 5th column for hp25C</p><p>* press-n-hold 3nd row, 5th column for hp21</p><p>* press-n-hold 4th row, 4th column for version info</p><p>* press-n-hold 1st row, 4th column to show greetings</p><p>* press-n-hold 1st row, 3rd column to edit greetings, use 0-9 to enter numbers and letters, Run/Stop to advance digit, Cls to abandon entry</p><p>* press-n-hold Pgm/Run toggle button to edit default startup settings, use &apos;F&apos; key to select rom, &apos;G&apos; key to select speed, Pgm/Run will save setting and exit setup</p><p>Schematic</p><p>_________</p><p><img src="https://cdn.hackaday.io/images/3036551439817188530.png"></p><br>
