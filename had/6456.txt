Essentially, it is an 8-channel relay controller with a wifi interface and an auxiliary RS-232 port for connection to industrial equipment over serial. The PCBs are designed as a dual layer stack with a 24-way FPC cable connecting the main control board to the relay board. This way I can swap out the ESP8266 based wifi board for something else (Ethernet/Zigbee/nRF) and retain the same baseboard. It is all designed to fit into a plastic DIN rail enclosure sourced from Aliexpress. Status LEDs for the relay channels are channeled through light pipes, through the top board, and out the front panel of the enclosure.

                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">I wanted to build a fairly simple project that combined a number of elements to further my skills. Mainly it was about creating a project to further develop my code writing skills for the ESP8266 in C code. <br /><br />Essentially, it is an 8-channel relay controller with a wifi interface and an auxiliary RS-232 port for connection to industrial equipment over serial. The PCBs are designed as a dual layer stack with a 24-way FPC cable connecting the main control board to the relay board. This way I can swap out the ESP8266 based wifi board for something else (Ethernet/Zigbee/nRF) and retain the same baseboard. It is all designed to fit into a plastic DIN rail enclosure sourced from Aliexpress. Status LEDs for the relay channels are channeled through light pipes, through the top board, and out the front panel of the enclosure. <br /><br />Microchip MCP23S08 GPIO expander over SPI for 8 GPIOs. TI TRS3221 for TTL->RS232</div>
                </div>

                        <div class="post-content details-content">
                            <p><p>General Features &amp; IOs:</p><ul><li>8x Relay Channels - 10A 250V AC with NO/NC/COMMON connections</li><li>PWR input (7-28V DC) with reverse power diode protection.</li><li>Internal DC-DC module with 5V output</li><li>RS232 level serial connection TX/RX/GND</li><li>Reset Button</li><li>Bootload/Program Button (GPIO0)</li><li>User Input Button</li><li>Status LEDs: Power, User Controlled (wifi activity), Serial TX, Serial RX</li></ul>
