                    <h5>Description</h5>
                    <div class="description">Goal was to create a cool new game console for Retro Gaming.<br />The new Raspberry Pie with the Retropie project gave the opportunity. I designed a small case in Autodesk 123D with a mechanically operated cover.  The case was printed in XT Polyester which is a bit nicer then PLA or ABS. The Pi B+ was added along with some leds, stepper motor, and a ATX power supply. It plays all old-school dos games like Kings Quest or Monkey Island and includes emulators for all popular consoles like NES, MEGADRIVE, PLAYSTATION etc etc. <br />The console is finished now and recently upgraded with the Raspberry Pi 2 model B.   </div>
