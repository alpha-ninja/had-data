                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>This <a target="_blank" href="https://hackaday.io/project/2236-flex-modules">Flex Module</a> uses the new MAX30100 module.  Requires a 3v3 VDD input for the LEDs.   Also requires a 1v8 power source, so we added a tiny SMPS onboard powered from the VDD.  1v8 SMPS is around 500mA, so the board provides it to auxiliary boards through a pin out.  We need to make the driver once we have a board ready.<br></p>
                                
                            </p>
