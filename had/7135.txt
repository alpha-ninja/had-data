                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">Taking Open Source Hardware to a new scale, the OSE Bulldozer will make serious earthmoving accessible to any community, enterprise or individual with access to a workshop and a few basic tools. <br /><br />This machine is part of OSE's Global Village Construction Set (GVCS), a development project that aims to open source  the 50 key technologies needed for a modern civilization, allowing for a radically decentralized and democratized economy. See my 2011 Ted Talk for more on this: <a href='http://www.ted.com/talks/marcin_jakubowski?language=en' target="_blank">http://www.ted.com/talks/marcin_jakubowski?language=en</a><br /><br />The Bulldozer enables the earthworks needed for sustainable construction and regenerative agriculture, from grading roads to laying foundations, from building ponds to subsoil plowing. Costing 5 - 10 times less than  commercial equivalents, this project will lower the very high barriers that exist to doing this kind of work. </div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p> We invite you to join us in creating  the first Open Source Bulldozer: contribute to the development process <a target="_blank" href="http://opensourceecology.org/wiki/bulldozer">here</a>, or come build the first prototype with us in our upcoming <a target="_blank" href="http://opensourceecology.org/bulldozer-workshop/">workshop</a>. </p><p>This project builds on the success of our <a target="_blank" href="http://opensourceecology.org/wiki/MicroTrac_2015">Microtrac</a> prototype, which forms the base module of the Bulldozer. You can get a taste of  <a target="_blank" href="https://vimeo.com/134305061">how we built it</a>,  download the <a target="_blank" href="https://3dwarehouse.sketchup.com/model.html?id=u151bd188-2bdc-49fb-90b8-02b665e7baa3">CAD file</a>, and see the <a target="_blank" href="https://player.vimeo.com/video/134894033">results</a> (moving 8,000 lbs in an initial test)<span class="redactor-invisible-space">.  </span> </p><p><strong>The OSE Bulldozer is: </strong><br></p><p>Fully Open Source<br>- collaborative design, anyone can contribute<br>- full documentation released, including detailed instructionals that make replication a realistic option<br>- CC-BY-SA license<br></p><p>Meets or exceeds industry standards<br>- 18,000 lb torque<br>- 4.4 mph speed<br>- 168 hp<br></p><p>Radically modular<br>- for scalability: 2,000lb Microtractor base module is stacked to make a 13,000lb bulldozer<br>- for flexibility: add or exchange modules<br>- for interoperability: modules are shared by other machines in the GVCS</p><p>Fuel agnostic<br>- the PowerCube engine module can be built to run on any fuel<br>- we&apos;ve already prototyped a charcoal gasifier which has successfully run the PowerCube on pyrolysed biomass from our farm<br></p><p>Easy to fabricate<br>- designed for manufacture with a minimal tool set<br>- built from simple assemblies of off-the-shelf parts<br></p><p>Designed for lifetime use<br>- all modules are accessible for repair or replacement.<br>- bolt-together, easily disassembled construction<br></p><p>Multi-functional<br>- unlimited potential to add new modules<br>- designed for easy attachment of new tools and functions anywhere on the machine<br></p><p>Part of a whole ecology of tools<br>- The GVCS includes Open Source versions of machines that can build the bulldozer, as well as tools that complement its functionality.<br>- For example: our Metalworker lowers the cost of building the Bulldozer. The Bulldozer can grade a foundation site for a building, then the Compressed Earth Brick Press creates the walls of the building from on-site soil. All of these tools are powered by the same PowerCube module. </p><p>We publish our design work online as we go - you can follow along on these two Google Docs: </p><p><a target="_blank" href="https://docs.google.com/presentation/d/1meNG3gaM6TywGse7i9j7CXE781x1uFtbg0W6KkXDVJM/edit#slide=id.g594c8e3af_1_0">Working Document</a></p><p><a target="_blank" href="https://docs.google.com/presentation/d/1sBipRXQeSxL3bGcT5ZUQ9ZlEyqAQtQ5SS_l86l5KQfA/edit#slide=id.g594c8e3af_1_0">Modules</a><br></p>
                                
                            </p>
