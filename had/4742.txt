        <meta name="twitter:description" content="In my professional life I have used many different gas analyzers for confined space entry as well as emergency response.  These units are very expensive and require the user to be able to read the display or be close enough to hear the alert generated if an unsafe condition exists where human entry would need other measures.

I made this to be a very cheap and easily reproduced product which can simply be thrown into any uncertain environment such as a confined space, building with a possible gas leak, fire etc.  Any potential entry person can just toss the ball into the area and remotely receive the gas and smoke levels via voice from several hundred meters away. 

 I used an MQ2 smoke, LPG (Liquified Petroleum Gas), and CO (Carbon Monoxide) sensor.  I used freely available code to get the sensor working with an Arduino Nano.  Then taking an Ultra Cheap 433MHZ transmitter to output real voice transmit to any basic radio within the area.

This project is fully open-source!  Enjoy">

                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">In my professional life I have used many different gas analyzers for confined space entry as well as emergency response.  These units are very expensive and require the user to be able to read the display or be close enough to hear the alert generated if an unsafe condition exists where human entry would need other measures.<br /><br />I made this to be a very cheap and easily reproduced product which can simply be thrown into any uncertain environment such as a confined space, building with a possible gas leak, fire etc.  Any potential entry person can just toss the ball into the area and remotely receive the gas and smoke levels via voice from several hundred meters away. <br /><br /> I used an MQ2 smoke, LPG (Liquified Petroleum Gas), and CO (Carbon Monoxide) sensor.  I used freely available code to get the sensor working with an Arduino Nano.  Then taking an Ultra Cheap 433MHZ transmitter to output real voice transmit to any basic radio within the area.<br /><br />This project is fully open-source!  Enjoy</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>Using the Arduino I was able to easily measure the gas levels.  Using the Talkie library I was able to then translate these values into true synthesized human voice (with NO special boards/modules)  The transmitter accepts the voice output and directly transmits it out on 433mhz to ANY receiver.  It is sending this as an AM transmission- but surprisingly any FM receiver such as scanners/handy talkie radio, EMS radios etc have no trouble receiving the clear voice values.</p><p>Here is the enitre video demo:</p><p><a href="https://www.youtube.com/watch?v=_83s2re38ZU&amp;list=UUTo55-kBvyy5Y1X_DTgrTOQ" target="_blank">Video Demo</a></p><iframe width="560" height="315" src="https://www.youtube.com/embed/_83s2re38ZU" frameborder="0" allowfullscreen=""></iframe><p>Updated video and code walk-through can be found in the links.  Further info can also be found in the GitHub repo which includes all the code, design files, PCB layout and design document.<br></p><p><img src="https://cdn.hackaday.io/images/5057611439478158205.JPG" width="403" height="281" style="width: 403px; height: 281px;"></p><p>Here is a demo I did on the Adafruit Show and Tell show.  Fast forward to <strong>09:46</strong> as the direct link doesn&apos;t seem to work here:</p><iframe style="width: 500px; height: 281px;" width="500" height="281" src="//www.youtube.com/embed/U16WR56cFHs" frameborder="0" allowfullscreen=""></iframe>
