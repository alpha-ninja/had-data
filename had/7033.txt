                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">MobileNode aims to be a mobile IoT device. Its GPS/GLONASS module makes possible to track the device. Its GSM/GPRS module makes possible to send/receive data.<br /><br />You can attach sensors such as: air quality (CO, CO2, O2, etc), temperature, humidity, fire, motion, etc. Also you can attach external devices such as lighting devices, servo motors and other electronic circuits. Thereby the MobileNode is able to monitor polution, reduce consumption of electricity, collect food data and control food production devices.<br /><br />Every MobileNode comes with an attached tag, which contains two keys: public and private keys. You can push data to the cloud by using your private key and you can access such data by using the public key. The GPS coordinates are shown in a Google Map including the device orientation data given by the geomagnetic sensor.<br />The main example sketch allows you to send data to your free cloud  account.<br />The MobileNode board senses the battery remaining charge.</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p><img src="https://cdn.hackaday.io/images/4499041438278616692.gif"><strong></strong></p><p>Included Atmel MCU @ 8MHz<br></p><p>Arduino compatible</p><p>The MCU includes a preprogramed Arduino compatible bootloader.<br></p><p>USB programming and charging</p><p>The MobileNode is programed via USB by default<br></p><p>Power button</p>Acceleration and geomagnetic sensor<p>ADCs, GPIOs and SPI</p><p><img src="https://cdn.hackaday.io/images/6223261438371378175.png"></p><p><strong></strong></p><p><strong></strong></p><p><strong>Small circular shape (7cm)</strong></p><p>The MobileNode PCB is designed to be as small as possible, following all the recomendations given by the parts manufacturers.</p><p><strong>Screw holes</strong></p><p>There are 4 holes for M1.2 or M1.4 screws. So it is possible to put the MobileNode inside a box or case.</p><p><strong>UART jumpers</strong></p><p>The UART ports of the GPS and GSM/GPRS modules can be connected externally, in order to debug. If the 4 jumpers are removed, the UART ports can be attached to an external device.</p><p><strong>Microphone</strong></p><p>MobileNode includes an electret microphone. You can hear all what is happening near to the MobileNode with a voice call.</p><p><strong>MicroSIM card</strong></p><p>The microSIM socket is small </p><p><strong>TVS diodes</strong></p><p>Since the MobileNode PCB is exposed to touch with hand, there are several TVS diodes, which protects sensitive parts such as the modules.</p><p><strong>GSM/GPRS module</strong></p><p>The M66 is a very small quad-band GMS/GPRS module which is able to handle TCP/ UDP/ PPP/ FTP/ HTTP/ SMTP/ CMUX/ SSL protocols.</p><p>This module is connected direclty to the battery, thereby it can measure the percent of its charge.</p><p><strong>Bluetooth 3.0</strong><br></p><p>The M66 also includes Bluetooth connectivity, specifically the SPP, OPP protocols.</p><p><strong>GPS Module</strong></p><p>The L86 has a multi-GNSS engine for GPS, GLONASS, and QZSS. The cold start is &lt;15s.</p><p><strong>On board antennas</strong></p><p>There is no need of attach external antennas to MobileNode, all of them are included in the board itself.</p><p><strong>External GPS antenna</strong></p><p>It is possible to connect an external antenna to the GPS module, since some aplications have to use the MobileNode indoor.</p><p><strong>Free cloud account</strong></p><p>Every MobileNode comes with an attached tag, which contains two keys: public and private keys. You can push data to the cloud by using your private key and you can access such data by using the public key. The GPS coordinates are shown in a Google Map including the device orientation data given by the geomagnetic sensor.</p><p><strong>Many code examples</strong></p>The main example sketch allows you to send data to your free cloud  account.
                                
                            </p>
