                    <h5>Description</h5>
                    <div class="description">A disco dance floor that uses capacitance sensors and a high-speed data bus to provide a fun and customizable interactive experience.<br /><br />I'm building this for Burning Man 2015. So it has to be cool, it has to be blinky, it should be interactive and it will need to stand up to the elements and silly people jumping on it! I've been working on it for the last 9 months and we're finally ready to go into full-scale production!  <br /><br />See the Details section for more info and a demo video of what we have so far!</div>

                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>Finally something that combines the floor from <a href="https://www.youtube.com/watch?v=LUID0jSh2Ic">Saturday</a><a href="https://www.youtube.com/watch?v=LUID0jSh2Ic"></a><a href="https://www.youtube.com/watch?v=LUID0jSh2Ic"></a><a href="https://www.youtube.com/watch?v=LUID0jSh2Ic"></a><a href="https://www.youtube.com/watch?v=LUID0jSh2Ic"></a><a href="https://www.youtube.com/watch?v=LUID0jSh2Ic"></a><a href="https://www.youtube.com/watch?v=LUID0jSh2Ic"></a><a href="https://www.youtube.com/watch?v=LUID0jSh2Ic"> Night Fever</a> and the footstep response from Michael Jackson's Billy Jean <a href="https://www.youtube.com/watch?v=Zi_XLOBDo_Y">music video</a>!</p><p>This will be a 12' x 12' interactive dance floor made up of 144 individually controlled cells, each with a hidden capacitive sensor powered by an Atmega and communicating with the master via an RS485 bus at 250Kpbs+.</p><p>The master controller is a node.js app which runs user submitted floor programs. This means you can control the entire floor with simple JavaScript programs! Want to play music when people step on certain squares? Sure. Turn the entire floor into <a href="https://en.wikipedia.org/wiki/Conway's_Game_of_Life">Conway's Game of Lfe</a>? Why not! </p><p>The app also doubles as an emulator that you can use to test your programs while the floor is offline.</p><p>You can find all the code (controller and slaves) in this github repo: <a href="https://github.com/jgillick/DiscoDanceFloor" target="_blank">https://github.com/jgillick/DiscoDanceFloor</a></p><p><iframe width="500" height="281" src="//www.youtube.com/embed/Z9SyA-YTFnU" frameborder="0" allowfullscreen=""></iframe><br></p>
