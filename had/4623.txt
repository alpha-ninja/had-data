                        <div class="post-content details-content">
                            <p><h2><p style="margin-top: 0px; color: rgb(90, 90, 90); font-family: 'Helvetica Neue', Helvetica, Arial, sans-serif; font-size: 14px; line-height: 20px; text-align: justify; background-color: rgb(250, 250, 250);"><img class="wp-image-216 size-full" src="http://omniacreator.com/wp-content/uploads/screenshots/omniacreator.png" alt="Omnia Creator" width="1170" height="522"></p></h2><h2>Code Editing</h2><p><img src="http://omniacreator.com/wp-content/uploads/screenshots/omniacreator2.png" alt="Omnia Creator"></p><p>Omnia Creator is powered by Qt Creator – the premier cross platform, free, and open source C/C++ IDE for professionals. Omnia Creator brings the following capabilities to your fingertips:</p><ul><li>File System, Bookmarks, Open Documents, and Code Outline Browser</li><li>Multiple Code Editors<ul><li>C/C++ Code Editor</li><li>Binary Editor</li><li>Diff Editor</li><li>Plain Text Editor</li></ul></li><li>Rapid Code Navigation Tools<ul><li>Switch Header/Source</li><li>Follow Symbol Under Cursor</li><li>Switch Between Function Declaration/Definition</li><li>Find Usages</li><li>Open Type Hierarchy</li><li>Open Include Hierarchy</li></ul></li><li>Syntax Highlighting and Code Completion<ul><li>Built-in Syntax Highlighting support for over 200 languages</li><li>Snippet Code Completion</li><li>Doxygen Code Completion</li></ul></li><li>Static Code Checking and Style Hints<ul><li>Underlines potential compile errors in red</li><li>Underlines potential compile warnings in green</li></ul></li><li>Code Folding<ul><li>Built-in support for over 200 languages</li></ul></li><li>Parenthesis Matching and Parenthesis Selection Modes</li><li>Record/Play/Save Macros</li><li>Text Editing<ul><li>Auto-indent Selection</li><li>Text Wrapping</li><li>Visualize Whitespace</li><li>Clean Whitespace on save</li></ul></li><li>Issue Flaging<ul><li>Marks compile errors in text editor</li><li>Marks compile warnings in text editor</li></ul></li></ul><ul><li>Advanced Find and Replace<ul><li>Search C/C++ Symbols, File System, Current File, and Open Documents</li><li>Regular Expression and Preserve Case when Replacing Support</li></ul></li><li>Code Editing<ul><li>Split Horizontally/Vertically</li><li>Open in new Window</li></ul></li><li>Code Refactoring<ul><li>Rename Symbol Under Cursor</li><li>Add Curly Braces</li><li>Move Declaration out of Condition</li><li>Rewrite Condition using another Operator</li><li>Split Declaration</li><li>Split If Statement</li><li>Swap Operands</li><li>Convert to Decimal, Hexadecimal, or Octal</li><li>Add Definition Inside Class, Outside Class, or in Implementation File</li><li>Add Function Declaration</li><li>Switch with Next/Previous Parameter</li><li>Extract Function</li><li>Extract Constant as Function Parameter</li><li>Add Local Declaration</li><li>Convert to Camel Case</li><li>Complete Switch Statement</li><li>Synchronize Function Declaration and Definition</li><li>Add #include for undeclared identifier</li><li>Reformat Pointers or References</li><li>Create Getter and Setter Member Functions</li><li>Move Function Declaration</li><li>Assign Function or Expression to Local Variable</li><li>Insert Inherited Pure Virtual Functions</li><li>Optimize For-loop</li></ul></li><li>Execute Custom Command-line Commands</li></ul><h2>Data Visualization</h2><p><img src="http://omniacreator.com/wp-content/uploads/features/oscilloscope_feature/oscilloscope.png" alt="Oscilloscope Window"></p>Serial Oscilloscope<p>Omnia Creator's Serial Oscilloscope Widget makes drawing line graphs, xy graphs, bar graphs, box plots, and scatter plots easy! Serial Oscilloscope let's you create an arbitrary number of graphs in a plotting window to display data from your microcontroller. You can set the plot name, axes labels, axes ranges, axes scale type (linear or logarithmic), background color, and more! You can also set each graph's name, line style, line color, fill style, fill color, and many more graph type specific settings.</p><p>Need a spectrum analyzer? Serial Oscilloscope can display the <a title="magnitude or phase" href="http://www.omniacreator.com/wp-download/libraries/InterfaceLibrary/html/ilshared_8h.html#ab8dbf9a634321b60688e48ec9b4f4a21">magnitude or phase</a> of the Fast Fourier Transform (FFT) of line graphs, xy graphs, and scatter plots. You can set the FFT plot axes labels, axes ranges, axes scale type (linear or logarithmic), and more! You can also set each FFT graph's <a title="windowing function" href="http://www.omniacreator.com/wp-download/libraries/InterfaceLibrary/html/ilshared_8h.html#a135ed00e776ab3f2e69897a1dca19735">windowing function</a> and many more graph type specific settings.</p><p>Serial Oscilloscope Widget Windows are fully interactive. You can zoom in, zoom out, and pan around in the plot space. Additionally, hovering your mouse over a plot point will show the X and Y coordinate information.</p><p>Need to share your plot? You can even modify exported JSON files and tweak the plot to suit your needs.
