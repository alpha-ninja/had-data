                        <div class="post-content details-content">
                            <p><p>The details are in FAQ format, answering and explaining the project in greater detail. </p><p>&gt; What is this project all about? </p><p style="margin-left: 20px;">The goal of WE- Online is to make education absolutely free, at lest till year 10 and to standardizes the education system throughout the world. Education which is not restricted by any boundary, cast or race. Its about time that we start acting like one species. </p><p>&gt; But what can WE-Online or Education solve?</p><p style="margin-left: 20px;">Education can solve almost everything. To name few, Illiteracy, poverty, unemployment, hunger, crime due to hunger and poverty, empowers people to take their own decision, spreads awareness, helps in family planning, world population etc.</p><p>&gt; Is that It? </p><p style="margin-left: 20px;">That&apos;s not it. This program will not only help parents and families who want to give their children at home education but also help parents with mentally &amp; physically challenged children to get at-home education. </p><p>&gt; What is WE-Online?</p><p style="margin-left: 20px;">WE-Online is a system put together by using WE-Pods, WE-Builds and WE-Site. All these three together can get any individual educated, for free. However you just need to have access to WE-Site if you are in a developed/developing country and who can afford a smart device i.e. no nonsense only business.</p><p>&gt; What are WE-Pods?</p><p style="margin-left: 20px;"> WE- Pods are Pods or Huts if you will, that will act as a portable stationary school. These pods will be built by waste/trashed caravans, containers or boxes that can be hacked and reused to accommodate a table, chair and WE-Builds. </p><p>&gt; What are WE-Builds?</p><p style="margin-left: 20px;"> WE-Builds are computers assembled out of hacked electronic waste/excess like desktops and laptops. There are tons of e-waste in various developed and developing countries. Using these we can build a very basic computer powerful enough to run just a browser and a webcam. The operating system will be Linux, mainly Ubuntu for its UI. This will not be limited to hacked broken devices, we will be working on raspberry pis and other open source devises to build a basic computer to deliver the service.</p><p>&gt; Where will you get the components? </p><p style="margin-left: 20px;">Its easy, initially we will be asking for donations of old laptops and computers. Now a days people change their smart devices within 2 years. Just by asking few of my friend and family I was able to collected 3 laptops that I can work on. Other source would be like donations / Prize money. </p><p>&gt;What is a WE-Sight? </p><p style="margin-left: 20px;">WE-Sight is simply WE-Online website that can be accessed by any one, anywhere and any time. The website would be more like MOOC and we are also planning to have live classroom sessions but that&apos;s a long way to go.  </p><p>************* still more to come, and further editing is required. If you have your own questions or doubts about this project please leave a message or a comment. Criticism are much appreciated. Thx***********************<br></p>

                        <h5>Build instructions</h5>
                        
                        
                            <ul class="post-content section-instructions-list">
                                
                                    <li>
                                        <span class="instruction-number">1</span>
                                        <div class="instruction-list-item"><p>First we would need to download the software that will help us make a bootable linux USB pen-drive. That software can be downloaded at this website called <a href="http://unetbootin.github.io/">unetbootin</a>. Make sure you download  the correct file according to your operating system.</p><p>* (license and term of use for this free software can be found <a href="http://unetbootin.github.io/#credits" target="_blank">here</a>.)  </p><p>Please make sure that you have at least 2 GB of space available on the USB stick.</p></div>
                                    </li>
                                
                                    <li>
                                        <span class="instruction-number">2</span>
                                        <div class="instruction-list-item"><p>Now download Linux ISO file i.e. the Operating system. I will be using Ubuntu as my OS because I like their UI and ton of support that is available on line. And the other reason is that it can be run on a very low end system.  </p><p>The ISO link can be found <a href="http://www.ubuntu.com/download/desktop" target="_blank">here</a>. While downloading make sure you read and downloading the right stuff according to your built/system configuration. </p><p>* (license and term of use for this free software can be found <a href="http://www.ubuntu.com/about/about-ubuntu/licensing" target="_blank">here</a>.)<br></p><p>At this moment of time this OS is only being used for personal use for this project. </p></div>
                                    </li>
                                
                                    <li>
                                        <span class="instruction-number">3</span>
                                        <div class="instruction-list-item"><p>Next open the Unetbootin software. </p><p>This software does not need any installation as such as it is functional on launch. </p><p><img src="https://cdn.hackaday.io/images/4658751439843721958.jpg"></p><p>Select the radio button at the bottom of the application window which says &apos;Diskimage&apos; select ISO in the drop down menu. </p><p>Then browse to the folder location where the ISO image has been saved. </p><p>Let the &apos;Space used to preserve files across reboots&apos; be &apos;0&apos; as we don&apos;t need it (unless you know what you are doing). </p><p>Select &apos;Type&apos; as &apos;USB Driver&apos; (we are making USB bootable drive, remember?) </p><p>Be careful on what you select your driver (don&apos;t use your C or D :p) </p><p>Then press OK. </p></div>
