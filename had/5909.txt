                    <h5>Description</h5>
                    <div class="description">Have you ever wanted to control your OpenVPN client daemon at the click of a button, as well as know whether it is running correctly or not from a glance? If you use this VPN Light Switch, you can do just that!</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>The OpenVPN client daemon is quite a versatile little dude. Unfortunately, if you use it on a WRT based router to enable it for all devices on your network, it can be quite difficult to control, expecially if you use devices that don&apos;t have web browers.</p><p>In that sense, the VPN Light Switch solves this issue by creating an easy to use interface that enables you to view your VPN status, as well as pause and resume it.</p><p>By using an ESP8266 module, the NodeLUA firmware and <a href="https://hackaday.io/project/5909-vpn-light-switch/log/18288-system-state-machine">enabling the management interface</a> on your VPN client daemon, you can create a very simple interface for interfacing to the daemon, in an all in one nature.</p><p>Here is a simple state machine for how the ESP module operates once connected to the Wi-Fi network that has the VPN service you want to control:<br></p><p><img src="https://cdn.hackaday.io/images/1424601432224471769.png">
                                
                                    <a class="readmore" href="/post/22865">Read more &raquo;</a>
                                
                            </p>
