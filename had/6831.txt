                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">Integrated scope can measure upto 3 different physical quantities such as temperature, voltage and current using dedicated sensors. With a Bandwidth of 200 MSPS it was tested to measure waveforms upto 20 MHz without significant distortion. It has MATLAB GUI support using which the user can communicate with the host cpu, this allows the user to carry out signal processing tasks such as Digital Filtering, spectrum analysis (FT) using the computational power of the cpu. It can also perform Math functions such as addition, subtraction and multiplication of the two waveforms.</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>The project can be divided into five subsystems as shown below. The functions of each of these subsystems is described below:-</p><p><img src="https://cdn.hackaday.io/images/7755051437379168594.jpg"></p><p><strong>1. Analog Front End</strong><strong>:</strong> It is made of various sensor modules which convert</p><p>physical quantities into electrical signals. Each sensor module is integrated</p><p>into respective measurement probe and the kind of physical quantity being</p><p>measured depends on the kind of probe connected to the scope. The</p><p>electrical signals from the sensor output is then attenuated and level shifted</p><p>so that it matches the input range of ADC. The attenuation factor depends on</p><p>the output range of the sensor connected to the scope. It is controlled by the</p><p>microcontroller with the help of analog multiplexers.</p><p><strong>2. Data Acquisition System:</strong> This subsystem is made of two High speed</p><p>ADC&#x2019;s, two MAX4203 buffers, two Dual DAC&#x2019;s and two Atmega32</p><p>microcontrollers. The ADC is used to sample electrical signals. After sampling</p><p>the values are stored in either atmega32 or in ADC internal capture buffer as</p><p>required. The two Dual DAC&#x2019;s along with the MAX4203 set the top and</p><p>bottom reference for the two ADC&#x2019;s.</p><p><strong>3. Main Microcontroller:</strong> The main microcontroller plays a central role in the</p><p>operation of integrated scope. It coordinates the operation of all other</p><p>devices. It takes input from Data Acquisition System, Control Panel and USB.</p><p>All these three inputs are used for rendering the image. As an example if the</p><p>user turns on the grid lines, the microcontroller will carry out a routine which</p><p>writes the grid lines on screen.</p><p><strong>4. Control Panel:</strong> The Control Panel uses three Atmega8 microcontrollers to</p><p>record the status of all switches, Rotary encoders and buttons used in the</p><p>control panel. Its primary function is to record the inputs given in control panel</p><p>and send the data to main microcontroller whenever required.</p><p><strong>5. LCD Screen</strong>: This scope uses a 480*272 4.3 Inch TFT LCD Module for</p><p>displaying the waveform and other relevant details as required. It is also</p><p>integrated with a touch screen interface which facilitate touch screen</p><p>operation.</p>
                                
                            </p>
                        </div>
