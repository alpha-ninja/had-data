                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">Iridium Eye will be a affordable quadcopter capable of creating high definition 3d maps of the world. Inspired by 2DRobotics, we aim to put 3d mapping into the hands on anyone who need it. We aim to simplify the skill set it normally takes to fly a quadcopter by using GPS assistance. Along with this we will use a simple open software to generate the 3d maps.<br /><br />Note: The current main image is from the Redditor rubiksman, who inspired me to do this project.</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p><strong>Why?</strong></p><p>3d mapping technology is godsend for gold miners as gold tends to accumulate in areas where rivers turn and dip and 3d maps would allow them to pinpoint the next mining location. Most small gold miners don't have access to mapping technology due to the required skill set and high cost of entry and we want to help.<br></p><p><strong>How does it make the 3d maps?</strong></p><p>The quadcopter uses GPS assistance to fly in a grid pattern and stabilise after the user designates the area to be mapped using computer software. A open program called Pix4D then stitches together the geotagged high-res rapid fire photos into a 1CM accurate point cloud.</p><p><strong>How is it affordable?</strong></p><p>Most mapping quadcopter use heavy DSLR cameras  set in a rapid fire mode to shoot the images. This raises the cost of the quadcopter to multiple thousands of dollars. With the rise of light, affordable 4K quality action cameras, we have made it possible to use a sub 1kg size drone to take a 4K video, which then can be converted into high resolution still images.</p>
                                
                            </p>
                        </div>
