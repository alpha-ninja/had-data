                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">LaserOscope (Build a Laser Oscilloscope Using Lego, littleBits, Erector Set, and the Kano Computer). In honor of The International Year of Light I’ll demonstrate how to use the Kano computer to drive a littleBits motor with an optical coupler, or optocoupler. The littlBits motor spins a mirror that projects a laser on a screen to visualize audio wave forms.</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>Follow me on Twitter: <a href="https://twitter.com/SteveSchuler20" target="_blank">https://twitter.com/SteveSchuler20</a>.<br></p><p>An optocoupler, <a href="http://en.wikipedia.org/wiki/Opto-isolator">according to Wikipedia</a>, "is a component that transfers electrical signals between two isolated circuits by using light." The Kano Computer is one isolated circuit and the littleBits Light Sensor/Motor is the second isolated circuit. I'll use the Kano Computer to flash an LED (Light Emitting Diode) and these flashes are read by the littleBits Light Sensor to turn the littleBits motor on and off. &lt;/p&gt;&lt;p&gt;While the LED of the Kano is switched on, the light sensor will switch the motor on and when the LED is switched off the light sensor will switch the motor off. The faster the LED is switched on and off, the faster the motor will spin. Conversely, the slower the LED is switched on and off, the slower the motor will spin.</p><p>To get the Kano Computer to flash an LED, you'll need to install the scratchgpio5 software on the Kano. For detailed instructions see my review of the Kano computer <a href="http://www.science20.com/square_root_of_not/kano_computer_in_a_word_wow-146511">here</a>. Also, if you're not familiar with littleBits, see my review <a href="http://www.science20.com/square_root_not/add_light_sound_and_motion_your_crafts_and_projects_littlebits-127136">here</a>. In my review of the Kano Computer, I demonstrated how to use MIT's Scratch graphical programming language on the Kano to flash an LED (the "Hello World!" program of hardware hacking). I'll again demonstrate how to flash an LED, but using a special variable in Scratch called "MotorA". Scratch automatically handles all of the programming for MotorA to produce a "pulse width modulation" (PWM) signal on Pin 11 of the Raspberry Pi. For a primer on pulse width modulation, see <a href="http://www.science20.com/square_root_not/snap_circuits_science_magnetic_stirrer-120834">this article</a>. I'll show you how to write a simple program to increase and decrease the flashing of an LED by pressing keys on the Kano keyboard.<br></p><p><strong>Create the MotorA variable in Scratch</strong><br></p><p>(<a href="http://scratch.mit.edu/projects/editor/?tip_bar=getStarted">You can create your own Scratch programs here</a>).<br></p><p>On the Kano, open up the Scratchgpio5 version of Scratch. In the Kano version of Scratch, click on Variables (on the web version click on Data). Click on the "Make a Variable" button and type "MotorA" in the textbox.</p><p><img src="http://content.science20.com/sites/all/modules/author_gallery/uploads/459993636-scratchmotora.png"></p><p>Click the "OK" button and Scratch will create all the motor control variables for you.</p><p><img src="http://content.science20.com/sites/all/modules/author_gallery/uploads/632315442-scratchmotoracontrols.png"></p><p>The following is the Scratch program I created to control the flashing of an LED:</p><p><img src="http://content.science20.com/sites/all/modules/author_gallery/uploads/668315718-scratchmotorcontrol.png"></p><p>The Scratch program is fairly simple:</p><p>WHEN THE GREEN FLAG ICON IS CLICKED<br>DISPLAY MESSAGE: "Press up arrow to increase speed, down arrow to decrease speed, or spacebar to exit."<br>SET MOTOR A TO 0 (Stop motor)<br>REPEAT <br>WAIT .01 SECS (Slows processing down enough to display one keypress at a time)<br>DISPLAY THE SPEED SETTING OF MOTOR A<br>IF UP ARROW KEY PRESSED<br>INCREASE MOTOR SPEED BY 1<br>IF DOWN ARROW KEY PRESSED<br>DECREASE MOTOR SPEED BY 1<br>IF MOTOR SPEED IS GREATER THAN 100<br>SET MOTOR SPEED TO 100<br>IF MOTOR SPEED IS LESS THAN ZERO<br>SET MOTOR SPEED TO 0<br>UNTIL SPACEBAR IS PRESSED<br>SET MOTOR SPEED TO 0<br>END PROGRAM</p><p><strong>Build the laser mirror spinner</strong><br></p><p><strong><br></strong></p><p>To be able to view the sound wave patterns we'll need to bounce the laser beam off a spinning mirror and then project the beam on to a screen. Here's how I built my laser mirror spinner out of Lego and littleBits:</p><p><img src="http://cdn.instructables.com/FZJ/NZE7/I760R12P/FZJNZE7I760R12P.MEDIUM.jpg"></p><p><img
