                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">My Dad has trouble pressing the tiny buttons on his TV remote, so I'm building him a custom remote with only a few buttons that he actually uses.  I imagine that my Dad's problem is not unique, so hopefully many others would benefit from this project as well.<br /><br />Currently the project is still on breadboard stage.  I envision the final version that I give to my Dad would be a 3D printed box containing a custom PCB.<br /><br />This will be the first project where I would need to be concerned about battery life.  I'm having fun learning about sleep and interrupts.</div>
