                    <h5>Description</h5>
                    <div class="description">The nrf24le1 SoC could be extremely useful for IoT applications. Many projects already use the nrf24l01 chip, which also needs a microchip  to run it. The le1 is the exact same radio combined with a microprocessor. I believe it hasn't been used as much because it isn't very easy to use. This project will address that problem. I plan on making 2 PCBs, one using the 24 pin le1 and breaking out all the pins while also making it usable permanently as a sensor node. The other will be used to program it. <br /><br />I have already completed the software side of this project for both Windows and Linux. I also wrote a Wiring library. <br /><br />Once completed, this will be a very inexpensive, small, and flexible base for IoT applications. IoT frameworks are fundamental to monitoring most anything; weather, house temperatures, air quality, green house conditions etc. </div>
                </div>

                                0 comments
                            </a>
                            </small>
                            <p><p>I have heavily borrowed from <a href="https://packagecontrol.io/packages/AVR">SublimeAVR</a>, a Sublime Text plugin that lets you use the AVR toolchain, and made the equivalent for the nrf24le1. There will be a Windows and Linux version. </p><p>The Windows version will include GNU Make and the programmer (which is a perl script) 'compiled' so that the end-user doesn't need to bother setting up a perl environment. It also puts the required include and lib directories in the project and a Makefile so that everything just works out of the box. </p><p>The Linux version is mostly the same, with a 'compiled' perl script that includes the dependencies so there won't be any fuss installing extras. Make won't be included since it's likely to be installed anyway. </p><p>That should mean that SDCC and Sublime Text will be the only required installs. And neither will require any tinkering. </p><p>After everything else is ready, I will put this on <a href="https://packagecontrol.io/">Package Control</a> so it will be easy to install. Until then, check it out at my <a href="https://github.com/justind000/ionode_sublime_plugin">repo</a>. </p>
