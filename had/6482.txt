        <meta name="twitter:description" content="From an engineering standpoint, Enigma Machines are fascinating. Not being able to procure an original one, the next best solution is to create a modern replacement. In order to make mass production of this project easy, the hardware portion of this project is very simple. It it just an Arduino UNO and a Seeed Studios Touchscreen LCD encased together,  all the magic is in the software.

This project accurately implements the Enigma I, M3 and M4 machines. The Uhr switch, a rarely described separate device that scrambled letters through the plugboard connections was implemented as well. The operation of this device is very intuitive, no menus are used in its configuration. All settings are changed by interacting with the wheels and plugs. 

Privacy is a universal human need (or right). This device helps people communicate securely. While mass communications encrypted with this device have been read by third parties before, measures can be taken to defend against previous attacks.">

                <div class="section section-description">
                    <h5>Description</h5>
                    <div class="description">From an engineering standpoint, Enigma Machines are fascinating. Not being able to procure an original one, the next best solution is to create a modern replacement. In order to make mass production of this project easy, the hardware portion of this project is very simple. It it just an Arduino UNO and a Seeed Studios Touchscreen LCD encased together,  all the magic is in the software.<br /><br />This project accurately implements the Enigma I, M3 and M4 machines. The Uhr switch, a rarely described separate device that scrambled letters through the plugboard connections was implemented as well. The operation of this device is very intuitive, no menus are used in its configuration. All settings are changed by interacting with the wheels and plugs. <br /><br />Privacy is a universal human need (or right). This device helps people communicate securely. While mass communications encrypted with this device have been read by third parties before, measures can be taken to defend against previous attacks.</div>
                </div>
            

            
                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>The initial inspiration to create this was the Franklin Heath Enigma Simulator for Android. <a href="https://play.google.com/store/apps/details?id=uk.co.franklinheath.enigmasim">https://play.google.com/store/apps/details?id=uk.co.franklinheath.enigmasim</a> and D. Rijmenants Enigma Simulator v7.0 <a href="http://users.telenet.be/d.rijmenants/en/enigmasim.htm">http://users.telenet.be/d.rijmenants/en/enigmasim.htm</a></p><p>To get an introduction to the enigma machine and its operation, the following links may be helpful:</p><p><a href="https://en.wikipedia.org/wiki/Enigma_machine" target="_blank">https://en.wikipedia.org/wiki/Enigma_machine</a><br></p><p><a href="http://www.cryptomuseum.com/crypto/enigma/" target="_blank">http://www.cryptomuseum.com/crypto/enigma/</a><span class="redactor-invisible-space"><br></span></p><p><span class="redactor-invisible-space">The following manual by D. Rijmenants<span class="redactor-invisible-space"> describes in exquisite detail the operation, internals, and real messages sent using the enigma machine.</span></span></p><p><span class="redactor-invisible-space"><a href="http://users.telenet.be/d.rijmenants/Enigma%20Sim%20Manual.pdf" target="_blank">http://users.telenet.be/d.rijmenants/Enigma Sim Manual.pdf</a><span class="redactor-invisible-space"></span></span></p><p>This project uses the three screen solution for configuring the enigma machine without the use of menus. The main screen is shown on power up, The machine can be used to encrypt and decrypt messages. To change the settings, the logo is touched and the open machine view is shown. The wheels can be changed and their ring positions set in a realistic way by dragging along the perimeter of the wheel. The plugboard is configured in a third screen by touching the bottom of the screen. Once ten (10) plugs are set, the Uhr switch is shown, this is also set by dragging in a circular motion. </p><p>A wood or clear acrylic case was created that completely encases the machine and exposes the USB and power connectors. The instruction sheets were recreated in the proper scale and can be glued to the inside of the lid. </p><p>A Houseworks 1131 hinge attaches the top of the lid to the base. A small chain can be used to prevent the lid from falling back.</p><p>The machine can be configured and used through the touchscreen LCD, but it can also be used through the USB cable as a virtual serial port. </p><p>While this encryption algorithm has been broken in the past, we can apply Bruce Schneier saying &quot;<a href="https://www.schneier.com/crypto-gram/archives/1998/1015.html" target="_blank">given enough rounds, pretty much anything is secure</a>&quot; and recommend at least double encrypting a text with different settings. This procedure was used for <a href="http://www.enigmaworldcodegroup.com/#!officer-to-officer-decrypting/c4ce" target="_blank">communications between submarine officers</a>. </p><p>The enigma reflector prevented a key encoding to itself. The use of double encryption solves that. </p><p>Go to:</p><p><a href="https://people.physik.hu-berlin.de/~palloks/js/enigma/enigma-u_v20_en.html">https://people.physik.hu-berlin.de/~palloks/js/enigma/enigma-u_v20_en.html</a></p><p>set the wheels to A C A T and paste the following text:<br></p><p>jjjjj jjjjj jjjjj jjjjj jjjjj jjjjj jjjjj jjjjj jjjjj jjjjj jjjjj jjjjj jjjjj jjjjj jjjjj jjjjj<span class="redactor-invisible-space"><br></span></p><p><span class="redactor-invisible-space">you will get the text below, notice there are no &quot;J&quot; in it.</span></p><p><span class="redactor-invisible-space">click on new session and set the wheels to A D O G and paste the text below:</span></p><p>ixvqb dfydf peqtz yetxf tplka mpbzn qlady swngd kgyki mlgqv vgfrr mavph fprds nqrac ynxnp bgzae<span class="redactor-invisible-space"><br></span></p><p><span class="redactor-invisible-space">you will get the following result, notice that on the sixth group, there are two j together. </span></p><p><span class="redactor-invisible-space">zylcc panfh ebhgl mffae csxnp kjjlv ifovp yplem wvqrg ksysd cthik nboba xwqyw qffsn olyqo tylwy<span class="redactor-invisible-space"><br></span></span></p><p><span class="redactor-invisible-space"><span class="redactor-invisible-space">To decrypt, click on new session, set the wheels to A D O G and paste the text above. Copy the result, which should start with ixv..., click new session, set the wheels to A C A T and paste it, a string of &quot;J&quot; should come out.</span></span></p><p><span class="redactor-invisible-space"><span class="redactor-invisible-space">This product is available for purchase in our tindie and e-bay stores:</span></span></p><p><a href="https://www.tindie.com/products/ArduinoEnigma/arduino-enigma-simulator-simulates-enigma-i-m3-and-m4-machines/">https://www.tindie.com/products/ArduinoEnigma/arduino-enigma-simulator-simulates-enigma-i-m3-and-m4-machines/</a><br></p><p><a href="http://www.ebay.com/sch/enigmaarduino/m.html">http://www.ebay.com/sch/enigmaarduino/m.html</a><span class="redactor-invisible-space"><br></span></p>
                                
                            </p>
