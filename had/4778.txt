                <div class="section section-details">
                    <h5>
                        
                            Details
                        
                    </h5>
                    

                    
                        <div class="post-content details-content">
                            <p><p>The project goal is create and distribute  a custom Rasbian based SDcard image to run Mozilla Webmaker tools on Raspberry Pi. The setup will use most minimal setup with no requirement of connectivity to Internet. The Raspberry Pi with SDcard and Wifi Dongle will be used for web literacy web maker events in Africa. </p><p>The Raspberry Pi Server will host Web maker tools like Thimble with learning templates. The users would need to connect the server with Wifi from their laptops or using school computers using local ethernet  connection. <br></p><p>The goal is finish the project and field test at upcoming MozFest East Africa (<a href="http://mozfestea.org/2015/" target="_blank">http://mozfestea.org/2015/</a>) in July, 2015.<strong></strong></p><p><strong>What is Mozilla Webmaker ? </strong></p><p>Mozilla Webmaker is Mozilla's educational initiative. Webmaker's goal is to "help millions of people move from using the web to making the web." As part of Mozilla's non-profit mission, Webmaker aims "to help the world increase their understanding of the web, take greater control of their online lives, and create a more web literate planet.<br><a href="https://webmaker.org/en-US/about"></a></p><p><a href="https://webmaker.org/en-US/about" target="_blank">https://webmaker.org/en-US/about</a></p><p><a href="https://blog.mozilla.org/blog/2012/05/22/introducing-mozilla-webmaker/" target="_blank"></a><a href="https://blog.mozilla.org/blog/2012/05/22/introducing-mozilla-webmaker/" target="_blank">https://blog.mozilla.org/blog/2012/05/22/introducing-mozilla-webmaker/</a><strong></strong></p><p><strong>What is architecture of this system? </strong><br></p><ul><li>Basic Rasbian OS</li><li>Basic networking setup, port forwarding with Avahi/ZeroConf</li><li>Wifi Captive Portal with Hostapd and dhcpd server. </li><li>Nginx Default web server with Nodejs backend with supervisord<strong></strong></li></ul><p><strong>How to use this ? </strong></p><ol><li>Download the 2.5GB image  <a href="https://db.tt/PNLgmOf9" target="_blank">https://db.tt/PNLgmOf9</a> <a href="https://dl.dropboxusercontent.com/u/50372706/Raspbian-webmaker-alpha.img.tar.gz" target="_blank" rel="nofollow">(Link is broken)  Please send me email if you need this file hitmanarky (at) gMaIL (dot) com <br></a> </li><li> Write the image to the SDCard </li><li>Boot the raspberry Pi with the SDCard </li><li>Connect the Raspberry Pi to your local ethernet network. (Optional: If you have USB Wifi dongle then you can connect to 'Webmaker' Wifi network.)</li><li>On your computer, Open browser and type '<a href="http://webmaker.local/" target="_blank" rel="nofollow"></a><a href="http://webmaker.local" target="_blank">http://webmaker.local</a>' You should see the Thimble App UI.</li></ol><p>Login Information: <br></p>Username: pi<p> Password: raspberry<br> </p><p>Wifi  Password: raspberry</p>
                                
                            </p>
